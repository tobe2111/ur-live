# 카카오 로그인 디버깅 가이드

## 🐛 현재 증상

1. ✅ 카카오 로그인 페이지로 이동 성공
2. ✅ 로그인 완료
3. ❌ 메인페이지로 이동하지만 localStorage 비어있음
4. ❌ 세션 정보 저장 안 됨

## 🔍 가능한 원인

### 1. 카카오 로그인이 비활성화되어 있음
**확인 방법:**
1. https://developers.kakao.com 접속
2. 앱 선택
3. 왼쪽 메뉴: **"제품 설정" > "카카오 로그인"**
4. **"카카오 로그인 활성화"** 상태 확인
   - ✅ 활성화 필요 (ON 상태여야 함)

### 2. 동의 항목이 설정되지 않음
**확인 방법:**
1. 카카오 로그인 페이지에서
2. **"동의 항목"** 섹션 찾기
3. 다음 항목들이 설정되어 있어야 함:
   - **닉네임** (필수)
   - **프로필 사진** (선택)
   - **카카오계정(이메일)** (선택)

### 3. 백엔드 에러 발생
**확인 방법:**
백엔드에서 에러가 발생하면 조용히 메인페이지로 리다이렉트됩니다.

## ✅ 해결 방법

### Step 1: 카카오 로그인 활성화 확인

**카카오 개발자 콘솔:**
```
내 애플리케이션 선택
↓
제품 설정 > 카카오 로그인
↓
"카카오 로그인 활성화" 설정
↓
상태: ON (활성화)
```

**스크린샷:**
```
┌─────────────────────────────────┐
│ 카카오 로그인                    │
├─────────────────────────────────┤
│ 카카오 로그인 활성화    [ON] ◉  │ ← 이것이 ON이어야 함
│                                  │
│ Redirect URI                     │
│ https://live.ur-team.com/...     │
└─────────────────────────────────┘
```

### Step 2: 동의 항목 설정

**카카오 개발자 콘솔:**
```
내 애플리케이션 선택
↓
제품 설정 > 카카오 로그인
↓
동의 항목
↓
필수 항목 설정
```

**필요한 설정:**
| 항목 | 설정 | 필수 여부 |
|------|------|-----------|
| 닉네임 | ✅ 필수 동의 | 필수 |
| 프로필 사진 | ✅ 선택 동의 | 선택 |
| 카카오계정(이메일) | ✅ 선택 동의 | 선택 |

### Step 3: 비즈니스 인증 (선택)

**일반 사용:**
- 비즈니스 인증 없이도 테스트 가능
- 단, 개발자 본인과 등록된 테스터만 로그인 가능

**실제 서비스:**
- 비즈니스 인증 필요
- 모든 사용자가 로그인 가능

## 🧪 테스트 방법

### 1. 브라우저 콘솔 확인
```
F12 → Console 탭
로그인 시도
에러 메시지 확인
```

### 2. Network 탭 확인
```
F12 → Network 탭
로그인 시도
/auth/kakao/callback 요청 확인
Response 확인
```

### 3. 예상되는 Response

**성공 시:**
```
Status: 302 (Redirect)
Location: https://live.ur-team.com/?login=success&session=XXX&userId=YYY
```

**실패 시:**
```
Status: 302 (Redirect)
Location: https://live.ur-team.com/?error=login_failed
```

## 🔧 추가 디버깅

로컬에서 테스트:
```bash

cd /home/user/webapp
pm2 logs webapp --lines 50

# 로그인 시도
# 콘솔에서 다음 로그 확인:
# - "Kakao login success: { userId: ..., nickname: ..., state: ... }"
# - "Redirecting to: https://..."
```

### 카카오 API 응답 확인

카카오 토큰 교환 실패 시:
```json
{
  "error": "invalid_grant",
  "error_description": "authorization code not found"
}
```

카카오 사용자 정보 조회 실패 시:
```json
{
  "msg": "this access token does not exist",
  "code": -401
}
```

**"카카오 로그인 활성화"가 OFF 상태일 가능성이 높습니다!**

확인 방법:
1. https://developers.kakao.com
2. 앱 선택
3. 제품 설정 > 카카오 로그인
4. **"카카오 로그인 활성화"가 ON인지 확인**

이것이 OFF면 로그인이 완료되어도 백엔드에서 사용자 정보를 가져올 수 없습니다!
