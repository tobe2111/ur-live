## 🎯 주요 기능 영역

### 1️⃣ 사용자(User) 기능
#### ✅ 완료된 기능
- 카카오 로그인/회원가입
- 마이페이지 (프로필 관리)
- 주문 내역 조회
- 배송지 관리
- 장바구니
- 결제 (토스페이먼츠)
- 상품 검색
- 라이브 시청
- 숏폼 커머스 (요고 스타일)

#### ⚠️ 개선 필요
- 리뷰 작성/관리 기능 미흡
- 찜하기/위시리스트 없음
- 포인트/적립금 시스템 없음
- 쿠폰 시스템 없음
- 알림 기능 미흡

---

### 2️⃣ 셀러(Seller) 기능
#### ✅ 완료된 기능
- 셀러 회원가입/로그인
- 사업자 정보 관리 (주소 찾기 API, 자동 하이픈)
- 상품 등록/수정/삭제
- 라이브 스트림 생성/관리
- 라이브 컨트롤 (상품 전환)
- 주문 관리 (104건 정상 표시)
- 송장번호 등록
- 세금계산서 관리
- 프로필 편집 (SNS 링크, Bio)
- 셀러 공개 페이지 (/s/:sellerId)
- 매출 통계

#### ⚠️ 개선 필요
- 주문 필터링/정렬 부족
- CSV 엑스포트 없음
- 일괄 작업 기능 없음
- 재고 관리 미흡
- 상품 순서 저장 (드래그앤드롭) 미구현
- 정산 내역 상세 조회 부족
- 알림 시스템 없음

---

### 3️⃣ 관리자(Admin) 기능
#### ✅ 완료된 기능
- 관리자 로그인
- 셀러 승인/관리
- 라이브 스트림 관리
- 주문 전체 조회
- 정산 관리 (통계, 일괄 완료)
- 커미션율 설정

#### ⚠️ 개선 필요
- 대시보드 통계 부족
- 사용자 관리 기능 부족
- 신고/문의 관리 없음
- 로그/감사 기능 없음
- 배너/프로모션 관리 없음

---

### 4️⃣ 라이브 커머스 기능
#### ✅ 완료된 기능
- 라이브 스트리밍 (LivePageV2)
- 실시간 채팅
- 상품 카드 표시
- 실시간 상품 전환 (3초 폴링)
- 시청자 수 표시
- 장바구니 담기

#### ⚠️ 개선 필요
- WebRTC 실제 스트리밍 없음 (현재 iframe/영상 URL)
- 채팅 필터링/신고 기능 부족
- 좋아요/하트 애니메이션 없음
- 실시간 알림 (New Product) 부족
- 라이브 예약 알림 없음
- 다시보기 기능 없음
