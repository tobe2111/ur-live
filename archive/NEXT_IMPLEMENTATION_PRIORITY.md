## 🎯 우선순위 1: 비즈니스 핵심 기능 (즉시 구현 필요)

### 1️⃣ **셀러 대시보드 & 통계** ⭐⭐⭐⭐⭐
**왜 중요한가?**
- 셀러가 매출, 주문, 재고를 실시간으로 확인해야 함
- 데이터 기반 의사결정 지원
- 셀러 만족도 향상 → 플랫폼 성장

**구현 항목:**
```typescript
// 1. 매출 통계 대시보드
- 오늘/이번 주/이번 달 매출
- 매출 추이 차트 (Recharts)
- 인기 상품 TOP 10
- 최근 주문 목록

// 2. 실시간 라이브 통계
- 현재 시청자 수
- 실시간 주문 알림
- 채팅 모니터링
- 상품 클릭률

// 3. 재고 알림
- 재고 부족 상품 (< 10개)
- 재입고 추천
- 품절 상품 목록
```

**예상 시간**: 1-2일  
**난이도**: 중  
**비즈니스 임팩트**: 🔥🔥🔥🔥🔥

### 2️⃣ **검색 & 필터링 강화** ⭐⭐⭐⭐⭐
**왜 중요한가?**
- 사용자가 원하는 상품을 빠르게 찾아야 구매 전환율 상승
- 현재 기본 검색만 가능 (필터 없음)

**구현 항목:**
```typescript
// 1. 고급 검색 기능
- 자동완성 (Debounce)
- 검색 히스토리
- 인기 검색어

// 2. 필터링
- 가격 범위 (슬라이더)
- 카테고리
- 정렬 (인기순, 가격순, 최신순)
- 브랜드 (셀러)

// 3. 검색 결과 최적화
- 페이지네이션 (무한 스크롤)
- 결과 카운트
- "결과 없음" 대체 추천
```

**예상 시간**: 1-2일  
**난이도**: 중  
**비즈니스 임팩트**: 🔥🔥🔥🔥🔥

### 3️⃣ **주문 관리 시스템 (셀러)** ⭐⭐⭐⭐⭐
**왜 중요한가?**
- 셀러가 주문을 처리하고 배송 정보를 입력해야 함
- 현재 주문은 생성되지만 셀러가 관리할 UI 부족

**구현 항목:**
```typescript
// 1. 주문 목록
- 상태별 필터 (결제 완료, 배송 중, 배송 완료)
- 검색 (주문번호, 고객명)
- 정렬 (최신순, 금액순)

// 2. 주문 상세
- 고객 정보
- 상품 정보
- 결제 정보
- 배송지 정보

// 3. 배송 처리
- 송장번호 입력
- 택배사 선택
- 배송 상태 업데이트
- 고객 알림 (카카오 알림톡)

// 4. 취소/환불 관리
- 취소 요청 승인/거부
- 환불 처리
- 취소 사유 관리
```

**예상 시간**: 2-3일  
**난이도**: 중-높음  
**비즈니스 임팩트**: 🔥🔥🔥🔥🔥

### 4️⃣ **상품 리뷰 & 평점** ⭐⭐⭐⭐
**왜 중요한가?**
- 구매 결정에 가장 큰 영향을 미침
- 신뢰도 향상 → 전환율 상승
- 현재 리뷰 시스템 없음

**구현 항목:**
```typescript
// 1. 리뷰 작성
- 별점 (1~5)
- 텍스트 리뷰
- 이미지 업로드 (선택)
- 구매 확정 후에만 작성 가능

// 2. 리뷰 조회
- 상품 상세 페이지에 표시
- 평점 평균 & 분포 차트
- 정렬 (최신순, 높은 평점순)
- 필터 (평점, 사진 포함)

// 3. 리뷰 관리 (셀러)
- 리뷰 답변
- 부적절한 리뷰 신고

// 4. DB 스키마
CREATE TABLE reviews (
  id INTEGER PRIMARY KEY,
  product_id INTEGER,
  user_id INTEGER,
  order_id INTEGER,
  rating INTEGER, -- 1~5
  content TEXT,
  images TEXT, -- JSON array
  created_at DATETIME
);
```

**예상 시간**: 2-3일  
**난이도**: 중  
**비즈니스 임팩트**: 🔥🔥🔥🔥

### 5️⃣ **위시리스트 (찜하기)** ⭐⭐⭐⭐
**왜 중요한가?**
- 사용자가 관심 상품을 저장 → 재방문율 상승
- 마케팅 타겟팅 데이터
- 간단하게 구현 가능

**구현 항목:**
```typescript
// 1. 찜하기 기능
- 상품 상세/목록에서 하트 버튼
- 로그인 필요
- 즉시 반영 (낙관적 업데이트)

// 2. 찜 목록 페이지
- 찜한 상품 그리드
- 삭제 기능
- 장바구니 담기

// 3. DB 스키마
CREATE TABLE wishlists (
  id INTEGER PRIMARY KEY,
  user_id INTEGER,
  product_id INTEGER,
  created_at DATETIME,
  UNIQUE(user_id, product_id)
);

// 4. API
POST /api/wishlists (찜 추가)
DELETE /api/wishlists/:id (찜 삭제)
GET /api/wishlists/:userId (찜 목록)
```

**예상 시간**: 1일  
**난이도**: 낮음  
**비즈니스 임팩트**: 🔥🔥🔥🔥

```typescript
// 1. 알림 타입

- 알림 센터 페이지

// 3. DB 스키마
CREATE TABLE notifications (
  id INTEGER PRIMARY KEY,
  user_id INTEGER,
  type TEXT, -- order, live, discount, coupon
  title TEXT,
  message TEXT,
  link TEXT,
  is_read INTEGER DEFAULT 0,
  created_at DATETIME
);
```

### 7️⃣ **쿠폰 시스템** ⭐⭐⭐⭐
**구현 항목:**
```typescript
// 1. 쿠폰 타입
- 정액 할인 (1000원 할인)
- 정률 할인 (10% 할인)
- 배송비 무료
- 최소 구매 금액 조건

// 2. 쿠폰 발급
- 관리자/셀러가 생성
- 코드 입력 또는 자동 발급

// 3. 쿠폰 사용
- 결제 시 적용
- 중복 사용 불가
- 사용 내역 저장

// 4. DB 스키마
CREATE TABLE coupons (
  id INTEGER PRIMARY KEY,
  code TEXT UNIQUE,
  type TEXT, -- amount, percent, free_shipping
  value INTEGER,
  min_order_amount INTEGER,
  valid_from DATETIME,
  valid_until DATETIME,
  max_use_count INTEGER,
  used_count INTEGER DEFAULT 0
);

CREATE TABLE user_coupons (
  id INTEGER PRIMARY KEY,
  user_id INTEGER,
  coupon_id INTEGER,
  is_used INTEGER DEFAULT 0,
  used_at DATETIME
```

```typescript
// 1. 이미지 업로드 API
```
