🔑 Firebase Private Key 변환 도구

Cloudflare Pages 환경 변수용 줄바꿈 변환

⚠️
중요 안내
Firebase Console에서 다운로드한 JSON 파일의 private_key 값을 복사하여 아래 입력란에 붙여넣으세요. 이 도구는 \n 문자열을 실제 줄바꿈으로 변환합니다.
✅ 클립보드에 복사되었습니다! 이제 Cloudflare Dashboard에 붙여넣으세요.

📖 사용 방법

  1. Firebase Console 접속:
    https://console.firebase.google.com/.../serviceaccounts/adminsdk
  2. "Generate new private key" 버튼 클릭 → JSON 파일 다운로드
  3. JSON 파일을 텍스트 에디터로 열기 → "private_key" 값 복사
    (따옴표와 \n 문자열이 포함된 상태로 복사)
  4. 위의 "입력" 텍스트 영역에 붙여넣기
  5. "🔄 변환하기" 버튼 클릭
  6. "📋 클립보드에 복사" 버튼 클릭
  7. Cloudflare Dashboard 접속:
    https://dash.cloudflare.com/ → Workers & Pages → ur-live → Settings → Environment variables → Production
  8. FIREBASE_PRIVATE_KEY 변수 편집 → 클립보드에서 붙여넣기 → Save
  9. Deployments 탭 → "Retry deployment" 클릭 → 5-10분 대기
  10. 테스트: https://live.ur-team.com 에서 카카오 로그인 시도

❌ 잘못된 형식 (변환 전):

"-----BEGIN PRIVATE KEY-----\nMIIEvAIBADANBgkqhkiG9w0BAQ...\n-----END PRIVATE KEY-----\n"

✅ 올바른 형식 (변환 후):

-----BEGIN PRIVATE KEY-----
MIIEvAIBADANBgkqhkiG9w0BAQEFAASCBKYwggSiAgEAAoIBAQDE/s3PltvNta+W
...
-----END PRIVATE KEY-----
