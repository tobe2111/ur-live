# 라이브 방송 설정

유어딜은 3가지 방송 방식을 지원합니다. 상황에 맞게 선택하세요.

## 방송 방식 비교

| 방식 | 기기 | 난이도 | 특징 |
|---|---|---|---|
| **YouTube Studio** | PC/웹 | ⭐ 쉬움 | 별도 앱 없이 브라우저에서 바로 |
| **OBS Studio** | PC | ⭐⭐ 보통 | 화면 전환, 자막, 다양한 소스 활용 가능 |
| **네이버 프리즘** | 모바일 | ⭐ 쉬움 | 스마트폰으로 언제 어디서나 방송 |

---

## 공통: 방송 시작 전

1. 셀러 대시보드 → **라이브 방송** 접속
2. **방송 정보 입력**: 제목, 설명, 판매 상품 선택
3. 공개 설정 선택 (공개 / 미등록 / 비공개)
4. 방송 방식 선택 후 **방송 만들기** 클릭

---

## YouTube Studio 방식

:::tip 추천 상황
빠르게 방송을 시작하고 싶을 때, 별도 소프트웨어 없이 방송할 때
:::

1. 유어딜에서 **YouTube Studio 열기** 클릭
2. YouTube 스튜디오 → 좌측 **라이브 스트리밍** 메뉴
3. **스트림 시작** 클릭
4. 유어딜이 자동으로 방송 시작을 감지합니다 ✓

---

## OBS Studio 방식

:::tip 추천 상황
화면 전환, 자막, BGM 등 전문적인 방송 연출이 필요할 때
:::

### 최초 1회 설정

1. [OBS Studio](https://obsproject.com) 설치
2. OBS → **설정** → **방송** 탭
3. 서비스: **사용자 지정** 선택
4. 유어딜 대시보드에서 **RTMP URL**과 **스트림 키** 복사 후 붙여넣기
5. **확인** 저장

### 이후 방송

- 유어딜에서 새 방송 생성 → 기존 RTMP 키 자동 재사용
- OBS에서 **방송 시작** 클릭만 하면 됩니다

---

## 네이버 프리즘 방식

:::tip 추천 상황
야외, 현장 방송 등 모바일 환경에서 방송할 때
:::

### 최초 1회 설정

1. [네이버 프리즘 라이브 스튜디오](https://prismlive.com) 앱 설치
2. 유어딜 대시보드에서 QR코드 스캔
3. 자동으로 RTMP URL과 키가 앱에 입력됩니다
4. 프리즘 → **라이브 시작**

:::info QR코드 설정
유어딜 대시보드의 QR코드를 스마트폰으로 스캔하면 RTMP 설정 페이지가 자동으로 열려 복붙 없이 설정됩니다.
:::

## 방송 중 기능

방송이 시작되면 유어딜 대시보드 **Step 3 — 라이브 중**에서:

- **소개 상품 전환**: 탭 한 번으로 현재 소개 중인 상품 변경
- **채팅 확인**: 실시간 시청자 채팅 모니터링
- **타임딜 시작**: 할인율, 수량, 시간 설정 후 즉시 실행
- **경매 시작**: 시작가, 최소 증가금액, 진행 시간 설정

## 방송 종료

대시보드 우상단 **방송 종료** 버튼 클릭 → YouTube 브로드캐스트 자동 종료 → 방송 리포트 확인
