# 셀러 시작 가이드

유어딜 셀러가 되어 라이브 방송으로 수익을 만들어보세요.

## 시작 전 준비물

| 항목 | 설명 |
|---|---|
| ✅ Google 계정 | YouTube 라이브 방송 연동용 |
| ✅ YouTube 채널 | 채널 인증 완료 상태 권장 |
| ✅ 판매할 상품 | 실물 상품 또는 맛집 식사권 |

:::info YouTube 라이브 조건
모바일(프리즘)로 방송하려면 구독자 **1,000명 이상** 채널이 필요합니다.  
PC(OBS)와 YouTube Studio 방식은 구독자 수 제한이 없습니다.
:::

## 1단계: 셀러 계정 등록

1. [유어딜 셀러 페이지](https://live.ur-team.com/seller/login)에 접속합니다.
2. **셀러 회원가입** 후 기본 정보를 입력합니다.
3. 사업자 정보 또는 개인 정보를 입력하고 승인을 기다립니다.

## 2단계: YouTube 계정 연동

셀러 대시보드 → **라이브 방송** → **YouTube 계정 연동하기**

Google OAuth로 YouTube 채널을 연결하면 RTMP 키가 자동 발급됩니다.  
**최초 1회만** 설정하면 이후 방송마다 자동으로 재사용됩니다.

## 3단계: 상품 등록

셀러 대시보드 → **상품 관리** → **상품 등록**

- 상품명, 가격, 재고 수량, 상품 이미지를 입력합니다.
- 맛집 식사권의 경우 바우처 코드 방식으로 등록합니다.

## 4단계: 첫 라이브 방송

→ [라이브 방송 설정 가이드](/docs/seller/live-setup)에서 자세한 방법을 확인하세요.

## 수수료 안내

| 항목 | 요율 |
|---|---|
| 기본 플랫폼 수수료 | **15%** |
| 결제 수수료 | 결제사 실비 적용 |
| 딜 포인트 충전 | 수수료 없음 (1원 = 1딜) |

## 정산 일정

- 정산 기준: 매월 말일 마감
- 지급일: 다음 달 10일
- 방법: 등록된 계좌로 자동 이체
