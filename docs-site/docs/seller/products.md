# 상품 관리

## 상품 등록

셀러 대시보드 → **상품 관리** → **+ 상품 등록**

### 필수 입력 항목

| 항목 | 설명 |
|---|---|
| 상품명 | 구매자에게 표시되는 이름 |
| 가격 | 판매 기본가 (딜 포인트 결제 가능) |
| 재고 | 판매 가능 수량 |
| 대표 이미지 | 라이브 화면에 표시되는 상품 이미지 |
| 카테고리 | 검색 및 필터 분류 |

### 맛집 식사권 등록

맛집 상품은 **바우처 방식**으로 등록합니다.
- 구매 완료 시 고객에게 바우처 코드 자동 발급
- 카카오맵 연동으로 매장 위치 자동 표시

## 라이브 중 상품 전환

방송 중 소개하는 상품을 실시간으로 변경할 수 있습니다.

1. 대시보드 라이브 화면 하단 **소개 상품 전환** 영역
2. 전환할 상품 탭 → 즉시 반영

시청자 화면의 상품 정보가 실시간으로 업데이트됩니다.

## 재고 관리

- 타임딜/경매 종료 시 낙찰 수량만큼 재고 자동 차감
- 재고 0이 되면 **품절** 표시 자동 적용
- 재고 부족 시 셀러 대시보드에 알림 발송
