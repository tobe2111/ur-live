# 정산 안내

## 수수료 구조

| 항목 | 요율 |
|---|---|
| 플랫폼 수수료 | **15%** |
| 셀러 정산액 | **85%** (판매금액 기준) |

예시: 10,000딜짜리 상품 판매 시  
→ 셀러 정산: **8,500딜 (8,500원)**

## 정산 일정

| 단계 | 시기 |
|---|---|
| 정산 마감 | 매월 말일 |
| 지급일 | 다음 달 10일 |
| 방법 | 등록 계좌 자동 이체 |

## 정산 확인

셀러 대시보드 → **정산 관리**에서 실시간 확인 가능:
- 기간별 매출 현황
- 수수료 차감 내역
- 정산 예정액
- 과거 정산 이력

## 세금계산서

사업자 셀러의 경우 매월 정산일에 세금계산서 자동 발행됩니다.

:::info 딜 포인트 정산
구매자가 딜 포인트로 결제한 금액도 동일하게 현금으로 정산됩니다.  
딜 포인트 1 = 1원으로 정산 처리됩니다.
:::
