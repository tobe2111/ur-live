# 경매 & 타임딜

방송 참여도를 높이고 빠른 판매를 이끄는 두 가지 핵심 기능입니다.

## ⚡ 타임딜

한정 시간·수량으로 진행하는 초특가 이벤트입니다.

### 설정 방법

방송 중 대시보드 → **타임딜** 버튼

| 항목 | 설명 |
|---|---|
| 상품 | 타임딜 적용 상품 선택 |
| 할인율 | 예) 30% 할인 |
| 수량 | 한정 판매 수량 |
| 시간 | 진행 시간 (초 단위, 예: 30초) |

### 시청자 화면

- 카운트다운 타이머와 남은 수량이 실시간 표시
- 수량 소진 또는 시간 만료 시 자동 종료

:::tip 효과적인 타임딜
방송 시작 직후 / 클라이맥스 순간에 활용하면 시청자 집중도가 급상승합니다.
:::

## 🔨 라이브 경매

시청자가 실시간으로 입찰하는 경매입니다.

### 설정 방법

방송 중 대시보드 → **경매 시작** 버튼

| 항목 | 설명 |
|---|---|
| 경매 제목 | 경매 상품 또는 이벤트명 |
| 상품 연결 | 낙찰 시 연결할 상품 (선택사항) |
| 시작가 | 최초 입찰 시작 금액 (딜 포인트) |
| 최소 증가금액 | 입찰마다 최소 올려야 하는 금액 |
| 진행 시간 | 경매 진행 시간 (초 단위) |

### 낙찰 처리

- 시간 만료 시 최고 입찰자 자동 낙찰
- 낙찰 금액은 딜 포인트로 즉시 차감
- 낙찰자에게 알림 발송 및 주문 자동 생성

:::info 경매 vs 타임딜
- **타임딜**: 정해진 할인가로 선착순 구매 → 빠른 판매에 유리
- **경매**: 입찰 경쟁으로 가격 결정 → 희귀/한정 상품에 유리
:::
