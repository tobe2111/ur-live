# 구매자 자주 묻는 질문

**Q. 회원가입 없이 방송을 볼 수 있나요?**  
A. 방송 시청은 가능하지만, 채팅·구매·경매 참여는 로그인이 필요합니다.

**Q. 딜 포인트는 현금으로 환불되나요?**  
A. 충전 후 7일 이내 전액 환불, 이후에는 미사용 잔액의 90%까지 환불됩니다.

**Q. 타임딜을 놓쳤어요. 다시 살 수 있나요?**  
A. 타임딜은 한정 이벤트로 재진행 여부는 셀러 재량입니다. 셀러 알림을 켜두면 다음 타임딜을 놓치지 않을 수 있습니다.

**Q. 바우처 유효기간이 지나면 어떻게 되나요?**  
A. 유효기간 만료 바우처는 고객센터를 통해 환불 처리가 가능합니다.

**Q. 경매 낙찰 후 취소할 수 있나요?**  
A. 경매 낙찰은 구매 확정으로 취소가 어렵습니다. 단, 상품 하자의 경우 고객센터로 문의해주세요.
