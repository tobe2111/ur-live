# 유어딜에 오신 것을 환영합니다 🎬

**유어딜(UR Deal)**은 인플루언서 라이브 방송 기반의 라이브 커머스 플랫폼입니다.

## 유어딜이란?

유어딜은 인플루언서와 시청자를 실시간으로 연결하는 **라이브 커머스 플랫폼**입니다.

- **구매자**는 라이브 방송을 보면서 실시간으로 채팅하고, 타임딜·경매에 참여하며 최저가로 쇼핑합니다.
- **셀러(인플루언서)**는 YouTube 계정 하나로 바로 라이브 방송을 시작하고 상품을 판매합니다.

## 핵심 기능

| 기능 | 설명 |
|---|---|
| 📺 **라이브 쇼핑** | 실시간 방송에서 채팅하며 상품 구매 |
| 🍽️ **맛집 공동구매** | 인플루언서 추천 맛집 식사권 최대 70% 할인 |
| ⚡ **타임딜** | 방송 중 한정 시간·수량 초특가 |
| 🔨 **라이브 경매** | 실시간 입찰로 최저가 낙찰 |
| 💎 **딜 포인트** | 1원 = 1딜, 충전하여 결제 및 후원 |
| 👥 **친구 초대 할인** | 3명 초대 시 10% 추가 할인 |
