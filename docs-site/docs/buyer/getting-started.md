# 구매자 시작 가이드

## 회원가입

1. [유어딜](https://live.ur-team.com)에 접속합니다.
2. 카카오 / Google 소셜 로그인으로 간편 가입합니다.
3. 닉네임과 기본 정보를 입력하면 바로 쇼핑 시작!

## 라이브 방송 입장

홈 화면 → **라이브** 탭에서 현재 진행 중인 방송 목록을 확인할 수 있습니다.

방송 입장 시:
- 실시간 채팅으로 셀러에게 질문
- 상품 정보 실시간 확인
- 타임딜·경매 참여
- 즉시 구매 버튼으로 빠른 결제

## 알림 설정

관심 있는 셀러를 팔로우하면 방송 시작 시 **카카오 알림톡**으로 즉시 알림을 받습니다.

셀러 프로필 페이지 → **알림 받기** 설정
