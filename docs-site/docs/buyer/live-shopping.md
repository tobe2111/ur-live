# 라이브 쇼핑

## 라이브 방송 참여 방법

1. 홈 화면 **라이브** 탭 → 방송 선택
2. 실시간 채팅창에 질문 입력
3. 상품 패널에서 **구매하기** 클릭
4. 딜 포인트로 즉시 결제

## 타임딜 참여

방송 중 셀러가 타임딜을 시작하면:
- 화면에 카운트다운 타이머 표시
- **지금 구매** 버튼으로 즉시 참여
- 수량 소진 또는 시간 만료 전에 빠르게 클릭!

:::tip 타임딜 알림
타임딜은 예고 없이 시작될 수 있습니다. 방송에 집중하세요!
:::

## 경매 참여

1. 경매 시작 시 화면에 입찰 패널 표시
2. 입찰 금액 입력 (딜 포인트)
3. **입찰하기** 클릭
4. 마감 시 최고 입찰자 자동 낙찰

:::info 입찰 취소
경매 중 본인 입찰은 취소할 수 없습니다. 신중하게 입찰하세요.
:::

## 후원하기

셀러를 응원하고 싶다면 **후원** 기능을 이용하세요.
- 최소 500딜부터 후원 가능
- 후원 금액에 따라 👑💎⭐ 등급 배지 획득
- 방송 채팅창에 후원 메시지 강조 표시
