# 딜 포인트

딜 포인트는 유어딜의 자체 결제 수단입니다. **1원 = 1딜**로 충전됩니다.

## 충전 방법

마이페이지 → **딜 포인트 충전**

- 충전 수수료 없음
- 최소 충전 금액: 1,000원
- 결제 수단: 카드, 계좌이체, 카카오페이

## 사용처

| 사용처 | 차감 시점 |
|---|---|
| 상품 구매 | 결제 완료 시 즉시 차감 |
| 셀러 후원 | 후원 즉시 차감 (최소 500딜) |
| 경매 낙찰 | 낙찰 즉시 차감 |
| 타임딜 구매 | 결제 완료 시 즉시 차감 |

## 포인트 적립

| 적립 조건 | 적립량 |
|---|---|
| 텍스트 리뷰 작성 | +50딜 |
| 사진 리뷰 작성 | +100딜 |
| 동영상 리뷰 작성 | +200딜 |

## 유효기간

충전한 딜 포인트의 유효기간은 **충전일로부터 5년**입니다.

:::caution 환불 안내
미사용 딜 포인트는 충전 후 7일 이내에 전액 환불 가능합니다.  
7일 이후에는 미사용 잔액의 90%까지 환불됩니다.
:::
