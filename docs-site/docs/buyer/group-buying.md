# 맛집 공동구매

인플루언서가 직접 추천한 맛집 식사권을 최대 70% 할인으로 구매하세요.

## 이용 방법

1. 라이브 방송에서 맛집 식사권 확인
2. **구매하기** → 딜 포인트로 결제
3. 마이페이지 → **바우처** 탭에서 코드 확인
4. 매장 방문 시 코드 제시

## 레스토랑 지도

홈 → **맛집 지도** 탭에서 구매 가능한 매장을 지도에서 확인할 수 있습니다.  
카카오맵 연동으로 위치, 영업시간, 길찾기를 바로 확인 가능합니다.

## 친구 초대 추가 할인

친구를 초대할수록 더 저렴하게 구매할 수 있습니다.

| 초대 인원 | 추가 할인 |
|---|---|
| 1~2명 | 기본 할인가 적용 |
| 3명 이상 | **+10% 추가 할인** |

### 초대 방법

상품 상세 페이지 → **친구 초대** 버튼 → 카카오톡으로 공유

:::tip 함께 사면 더 이득
같은 맛집을 가려는 친구들과 함께 구매하면 모두 10% 추가 할인 혜택을 받습니다.
:::

## 바우처 유효기간

- 구매일로부터 **60일** 이내 사용
- 유효기간 만료 전 마이페이지에서 알림 발송
