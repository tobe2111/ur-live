유어딜에서 라이브 방송을 시작하는 방법은 세 가지입니다. YouTube Studio, OBS Studio, 네이버 프리즘.

오늘은 **OBS**와 **프리즘** 설정을 처음 하는 분들을 위해 단계별로 설명합니다.

<!-- truncate -->

## 먼저: 유어딜에서 방송 생성하기

OBS든 프리즘이든, 먼저 유어딜 대시보드에서 방송을 생성해야 합니다.

1. [셀러 대시보드](https://live.ur-team.com/seller) → **라이브 방송**
2. 방송 제목, 판매 상품 선택
3. 방송 방식에서 **OBS Studio** 또는 **네이버 프리즘** 선택
4. **방송 만들기** 클릭

방송이 생성되면 **RTMP URL**과 **스트림 키**가 발급됩니다.

:::info 최초 1회 설정의 비밀
RTMP 키는 한 번 설정하면 유어딜이 저장합니다. 다음 방송부터는 새 키를 받을 필요 없이 OBS/프리즘에서 방송 시작만 누르면 됩니다.
:::

## OBS Studio 설정 (PC 방송)

### 설치

[obsproject.com](https://obsproject.com)에서 무료로 다운로드하세요. Windows, Mac, Linux 모두 지원합니다.

### RTMP 설정 (최초 1회)

1. OBS 실행 → 우하단 **설정** 클릭
2. 좌측 **방송** 탭 선택
3. 서비스: **사용자 지정** 선택
4. **서버**: 유어딜 대시보드의 RTMP URL 붙여넣기
5. **스트림 키**: 유어딜 대시보드의 스트림 키 붙여넣기
6. **확인** 저장

### 권장 화질 설정

OBS → 설정 → **출력** 탭:

| 항목 | 권장 값 |
|---|---|
| 인코더 | NVENC (GPU) 또는 x264 (CPU) |
| 비트레이트 | 4,000~6,000 Kbps |
| 키프레임 간격 | 2 |
| 해상도 | 1920x1080 |
| FPS | 30 |

### 방송 시작

유어딜에서 새 방송 생성 → OBS 우하단 **방송 시작** 클릭 → 유어딜이 자동으로 감지하여 라이브 전환

## 네이버 프리즘 설정 (모바일 방송)

### 앱 설치

App Store / Google Play에서 **PRISM Live Studio** 검색 후 설치.

### QR 코드로 간편 설정

유어딜 대시보드 → 방송 생성 → **네이버 프리즘** 선택 → QR 코드 표시

스마트폰으로 QR 코드 스캔 → 설정 페이지가 자동으로 열립니다.

### 수동 설정 (QR 코드 없이)

1. 프리즘 앱 실행
2. **+** 버튼 → **Custom RTMP** 선택
3. RTMP URL과 스트림 키 입력
4. 저장 후 **Go Live** 클릭

### 프리즘 촬영 팁

- **조명**: 창문을 정면으로 두거나 링라이트 사용
- **거치대**: 흔들림 없는 안정적인 거치대 필수
- **배경**: 깔끔하고 브랜드 느낌이 나는 배경 준비
- **인터넷**: Wi-Fi 연결 권장 (업로드 10Mbps 이상)

## OBS vs 프리즘 선택 기준

| 상황 | 추천 |
|---|---|
| 실내 쇼룸, 스튜디오 방송 | OBS Studio |
| 야외, 현장, 식당 방송 | 프리즘 |
| 화면 전환, 자막, 배너 활용 | OBS Studio |
| 빠르게 방송 시작하고 싶을 때 | 프리즘 |
| PC 없이 혼자 방송 | 프리즘 |

두 가지 모두 익혀두면 상황에 따라 유연하게 활용할 수 있습니다.
