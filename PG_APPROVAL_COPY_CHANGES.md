## 🎨 디자인 변경 사항

### 색상 변경
- **LIVE 배지**: `bg-[#ff3b30]` (빨간색) → `bg-[#007aff]` (파란색)
- **아이콘**: `Circle` (애니메이션 점) → `Play` (재생 아이콘)

### 애니메이션 제거
- **배지 애니메이션**: `animate-pulse` 클래스 제거
- **"실시간" 느낌 제거**: 정적 영상 강조

## 🔄 되돌리기 방법

PG사 승인 후 원래 "라이브" 문구로 되돌리려면:

```bash
cd /home/user/webapp
git log --oneline -5
# 0c3b301 feat: Update homepage copy from live to video shopping for PG approval
# 이전 커밋으로 되돌리기
git revert 0c3b301
npm run build
npx wrangler pages deploy dist --project-name toss-live-commerce
```

또는 수동으로 변경:

1. `/home/user/webapp/src/pages/HomePage.tsx` 파일 열기
2. 아래 키워드를 찾아 변경:
   - `유어 쇼핑` → `유어 라이브`
   - `쇼핑 영상` → `라이브`
   - `영상으로` → `라이브로`
   - `영상 쇼핑` → `라이브 진행 중`
   - `YouTube & TikTok 영상` → `실시간 라이브 방송(YouTube & TikTok)`
   - `영상 쇼핑 시작하기` → `라이브 보러가기`
   - `추천 영상 쇼핑` → `지금 라이브 중`
   - `bg-[#007aff]` → `bg-[#ff3b30]` (배지 색상)
   - `<Play />` → `<Circle ... animate-pulse />` (아이콘)

## 🎯 PG사 제출 시 강조할 포인트

### 서비스 설명
```
"YouTube 및 TikTok의 정적 영상 콘텐츠를 활용한 영상 커머스 플랫폼입니다.
판매자가 업로드한 YouTube Shorts 또는 TikTok 영상을 통해 
상품을 소개하고 구매자가 영상을 시청하면서 편리하게 상품을 구매할 수 있습니다."
```

### 주요 특징
1. **정적 영상 콘텐츠**: YouTube 및 TikTok에 업로드된 기존 영상 활용
2. **일반 이커머스와 동일**: 장바구니 → 주문 → 결제 흐름
3. **실시간 방송 없음**: 미리 촬영/편집된 영상만 사용
