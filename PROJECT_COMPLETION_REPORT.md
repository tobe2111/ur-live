## 📊 프로젝트 개요

### 플랫폼 정보
- **유형**: B2C 라이브 커머스 플랫폼
- **타겟**: 한국 + 글로벌 시장
- **주요 기능**: 라이브 스트리밍, 실시간 상품 판매, 주문/결제, 정산 관리
- **기술 스택**: 
  - Frontend: React 18 + TypeScript + Vite
  - Backend: Cloudflare Workers + Hono Framework
  - Database: Cloudflare D1 (SQLite)
  - Deployment: Cloudflare Pages
  - Payment: PortOne (KR) + Stripe (Global)
