# UR Live 운영 비용 분석 (2026년 기준)

## 🏗️ 현재 인프라 구성

### 1. Cloudflare Pages + Workers
- **프론트엔드**: Cloudflare Pages
- **백엔드**: Cloudflare Workers (Hono 프레임워크)
- **데이터베이스**: Cloudflare D1 (SQLite 기반)
- **캐시**: Cloudflare KV (3개 네임스페이스)
  - SESSION_KV: 세션 관리
  - CACHE_KV: 일반 캐시
  - LIVE_CACHE: 라이브 스트림 캐시

### 2. Firebase (Google Cloud)
- **Authentication**: 이메일/카카오 로그인
- **Realtime Database**: 라이브 채팅 (Asia Southeast 1)
- **Storage**: 프로필 이미지 (사용 중)

### 3. 외부 서비스
- **Toss Payments**: 결제 처리 (트랜잭션 수수료만)
- **Resend**: 이메일 발송 (선택적)
- **Sentry**: 에러 모니터링

### 2. Cloudflare D1 읽기 최적화
**문제**: 대규모 유저에서 D1 읽기 비용 급증

**해결책**:
```typescript
// ✅ KV 캐시 활용 (현재 일부 적용 중)
const cacheKey = `product:${productId}`;
let product = await c.env.CACHE_KV.get(cacheKey, 'json');

if (!product) {
  // D1 조회 (캐시 미스)
  product = await c.env.DB.prepare('SELECT * FROM products WHERE id = ?')
    .bind(productId).first();
  
  // KV 캐시 저장 (1시간 TTL)
  await c.env.CACHE_KV.put(cacheKey, JSON.stringify(product), { expirationTtl: 3600 });
}

// 🎯 추가 최적화:
// 1️⃣ 자주 읽는 데이터 KV 캐싱 (상품, 스트림, 배너)
// 2️⃣ 스트림 목록 캐싱 (현재 LIVE_CACHE 사용 중)
// 3️⃣ 사용자 세션 KV 저장 (DB 조회 줄이기)
```

**절감 효과**: 100,000 MAU 기준 $3,000 → $500 (**83% 절감**)
