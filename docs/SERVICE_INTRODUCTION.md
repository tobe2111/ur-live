# 유어딜 (YourDeal) 서비스 소개서

> 라이브 커머스와 커뮤니티를 하나로 — 크리에이터와 소비자를 직접 연결하는 플랫폼

---

## 서비스 개요

**유어딜(YourDeal)**은 라이브 스트리밍 기반의 쇼핑 커머스 플랫폼입니다.
셀러(크리에이터·소상공인)가 YouTube·TikTok을 통해 실시간 방송을 진행하며 상품을 판매하고, 소비자는 방송 시청 중 즉시 구매·후원·경매 참여가 가능합니다.

식당 이용권 공구(그룹바잉), 뷰티/펫/숙박/액티비티 바우처 등 O2O 상품군도 운영하여 온라인-오프라인을 잇는 커머스 생태계를 구축합니다.

---

## 핵심 가치

| 가치 | 설명 |
|---|---|
| **실시간 연결** | 셀러-소비자 간 라이브로 소통하며 쇼핑 |
| **투명한 수수료** | 거래마다 정산 내역 공개, 셀러 수익 보장 |
| **쉬운 시작** | 별도 앱 설치 없이 웹 브라우저에서 방송·구매 가능 |
| **공정한 생태계** | 대형 셀러·소형 셀러 모두 노출 기회 균등 |

## 사용자 유형

### 1. 소비자 (User)
- 카카오 로그인(국내) 또는 이메일/구글(글로벌)
- 상품 탐색·구매, 라이브 시청·후원, 식사권/바우처 사용
- 그룹바잉 참여, 경매 입찰, 포인트(딜) 충전·사용
- 쿠폰, 위시리스트, 리뷰, 추천인 리워드

### 2. 셀러 (Seller)
- 크리에이터 또는 소상공인 누구나 신청 가능
- YouTube·TikTok 채널 연동, OBS·브라우저 방송 지원
- 상품 등록·재고 관리, 주문 처리·배송
- 라이브 중 후원 수령, 경매 진행, 쿠폰 발행
- 실시간 매출 분석, 정산 요청

### 3. 에이전시 (Agency)
- 복수 셀러를 관리하는 MCN·에이전시
- 멤버 셀러 실적 통합 관리, 그룹 캠페인 운영
- PK 배틀(셀러 간 방송 대결), 프로모트 부스트(노출 증대)
- 인센티브 구조 설계, 그룹 정산

### 4. 관리자 (Admin)
- 셀러 가입 심사·승인
- 상품·콘텐츠 모니터링, 어뷰징 탐지
- 정산 검토·집행, 쿠폰·배너 운영
- 시스템 모니터링, 플랫폼 설정

## 주요 기능

### 라이브 커머스

#### 라이브 방송
- YouTube Live 채널 연동 (RTMP 키 자동 발급)
- OBS·브라우저(WebRTC) 두 가지 방송 방식 지원
- 방송 예약 → 팔로워 알림 → 라이브 시작 원스텝
- 실시간 채팅 (시청자·셀러·관리자 역할 구분)
- 방송 중 상품 핀, 쇼핑 링크 즉시 삽입

#### 후원 (Donation)
- 라이브 중 소비자가 딜(포인트)로 후원
- 후원 리더보드 실시간 표시 (이름 마스킹 처리)
- 후원금 셀러 수령 → 10일 홀드 후 정산
- 어뷰징 방지: 자기 방송 후원 차단, Turnstile CAPTCHA, 일일 한도 5천만원

#### 경매
- 방송 중 한정 상품 실시간 경매 (기본 180초)
- 입찰 시 잔액 임시 보류(Hold), 종료 후 낙찰자만 결제
- 최소 입찰 단위 설정, 자동 낙찰 처리

---

### 커머스

#### 상품 쇼핑
- 카테고리·검색·추천 상품 탐색
- 상품 옵션(사이즈·색상 등), 재고 실시간 반영
- 즉시구매·장바구니 통합 결제 (Toss Payments)
- 배송 조회, 구매 확정, 리뷰 작성

#### 그룹바잉 (공구)
| 카테고리 | 예시 |
|---|---|
| 식사권 | 레스토랑, 프랜차이즈 식당 이용권 |
| 뷰티권 | 헤어살롱, 스파, 피부과 |
| 헬스권 | 헬스장, PT, 요가 |
| 펫권 | 미용, 동물병원, 간식 |
| 숙박권 | 호텔, 펜션, 게스트하우스 |
| 액티비티권 | 방탈출, 클라이밍, 요리 클래스 |

- 목표 인원 달성 시 공구 성사, 미달 시 자동 환불
- 카카오 맵 연동으로 주변 식당 지도 탐색
- 바우처 코드 → QR·PIN 방식 사용

#### 포인트 (딜)
- 1원 = 1딜, 토스 결제로 충전
- 후원·상품 결제 시 딜 차감
- 최소 후원 500딜

## 수수료 구조

| 항목 | 기본 수수료 |
|---|---|
| 일반 상품 판매 | 10% (셀러별 조정 가능) |
| 라이브 후원 | 10% |
| 식사권/바우처 | 5% |
| 카카오 알림톡 | 8원/건 (크레딧) |

정산 주기: 배송 완료 확인 후 영업일 기준 처리. 후원금은 10일 홀드 후 정산 가능.

## 보안 기준

- HTTPS 전용 (Cloudflare SSL)
- HttpOnly + Secure 세션 쿠키
- 관리자/셀러 쿠키 SameSite=Strict (CSRF 방어)
- 금융 액션(정산·환불) 어드민 감사 로그
- 결제 금액 서버 재계산 (클라이언트 값 불신)
- 봇 차단: Turnstile CAPTCHA (후원 등 민감 액션)
- Rate Limit: 로그인·결제·후원 등 엔드포인트 적용
- Sentry 에러 모니터링 (프론트 + 서버)
