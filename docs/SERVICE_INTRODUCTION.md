# 유어딜 (UR DEAL) — 서비스 소개

> **"라이브 방송 + 후원 + 경매 + 공구를 한 화면에서. 셀러는 5분 만에 시작, 소비자는 팬덤 경험으로 산다."**

유어딜은 한국형 라이브커머스 플랫폼이다.
셀러 진입 장벽을 ZERO 로 낮추고, 소비자에게는 "쇼핑 + 콘서트 + 경매" 가 결합된 경험을 제공한다.
대형 마켓플레이스(쿠팡·11번가)도, 단순 라이브 앱(그립·카카오쇼핑라이브)도, 단일 채널(YouTube Shopping) 도 줄 수 없는 가치를 만든다.

---

## TL;DR — 30초 요약

| 대상 | 핵심 가치 |
|---|---|
| **셀러** | 카카오 로그인 → 5분 내 첫 방송. 상품 + 후원 + 경매 + 공구 = 한 플랫폼 4개 수익원. 정산 D+5일 (업계 최단). |
| **에이전시 (MCN)** | 멤버 셀러 통합 대시보드, PK 배틀로 트래픽 폭증, AI 셀러 매칭, 정산 자동 일괄 관리. |
| **소비자** | 라이브 경매 낙찰의 짜릿함, 식사권 공구 카카오 바이럴, 좋아하는 셀러에게 직접 후원·인사받는 팬덤 경험. |
| **투자자** | 한국 시장 PMF: 카카오 OAuth + 토스 결제 네이티브 + 식당 O2O = Pinduoduo 가 못 한 한국 로컬 침투. |

## 1. 우리는 무엇이 다른가 — 차별점

### vs **그립 / 카카오쇼핑라이브** (라이브커머스 1세대)

| 항목 | 그립 / 카카오쇼핑라이브 | **유어딜** |
|---|---|---|
| 셀러 진입 | 별도 앱 설치 + 심사 + 보증금 | **브라우저만** OBS 송출, 카카오 로그인 5분 |
| 후원 시스템 | 없음 (쇼핑만) | **딜 포인트 후원 내장** — 라이브 화면에 닉네임 노출 |
| 경매 | 없음 | **실시간 경매** — 가격이 내려가다가 누군가 낚아챔 |
| 식사권/오프라인 | 없음 | **그룹바이 식사권** 카테고리 독자 운영 |
| 정산 | D+15~30 | **상품 D+5, 후원 D+10** |

> 그립은 "TV홈쇼핑의 라이브화", 유어딜은 "트위치+쿠팡+카카오선물의 결합".

### vs **YouTube Shopping**

| 항목 | YouTube Shopping | **유어딜** |
|---|---|---|
| 정산 | 월 단위, 채널 페이아웃 통합 | **상품별 D+5일, 투명 즉시 가시화** |
| 결제 | 글로벌 카드 위주 | **토스·카카오페이 네이티브** + 한국 가상계좌 |
| 오프라인 연계 | 없음 | **식당 O2O** — 식사권 발급/사용까지 한 앱 |
| 한국 셀러 진입 | 채널 1000명 + 4000시간 + 심사 | **카카오 로그인 즉시** |
| 후원/도네이션 | Super Chat (라이브 한정) | **모든 셀러 페이지에서 상시 후원** |

> YouTube Shopping 은 "콘텐츠 부가수익", 유어딜은 "커머스가 본업".

### vs **티몬 / 위메프 슈퍼딜**

| 항목 | 슈퍼딜 | **유어딜** |
|---|---|---|
| 본질 | 가격 할인 게시판 | **라이브 + 후원 + 경매 = 팬덤 경험** |
| 셀러-소비자 관계 | 익명 트랜잭션 | **닉네임 후원, 셀러 직접 감사 인사** |
| 시간 압박 | 24시간 한정 할인 | **방송 중 N분 한정 경매** — 즉시성 극대화 |
| 충성도 | 가격이 떠나면 떠남 | **셀러 팬덤 락인** |

> 슈퍼딜은 "싼 거 사러 가는 곳", 유어딜은 "좋아하는 셀러 보러 가서 사는 곳".

### vs **Naver Smart Store / 쿠팡**

| 항목 | 스마트스토어/쿠팡 | **유어딜** |
|---|---|---|
| 디스커버리 | SEO·광고 입찰 (CPC 부담) | **라이브 + 알고리즘 추천 + 셀러 팬덤** |
| 셀러 차별화 | 가격·리뷰 경쟁 | **방송 콘텐츠 + 후원 + 경매로 차별화** |
| 광고비 | 매출의 5~15% 광고 의존 | **방송 자체가 마케팅** |

> 쿠팡은 "검색해서 사는 곳", 유어딜은 "재미있어서 들렀다 사는 곳".

### vs **Pinduoduo (중국 공동구매)**

| 항목 | Pinduoduo | **유어딜** |
|---|---|---|
| 바이럴 채널 | WeChat | **카카오톡 공유** 네이티브 |
| 카테고리 | 저가 일반 상품 위주 | **식사권/오프라인** 카테고리 독자 |
| 라이브 결합 | 별도 앱 분리 | **공구 + 라이브 + 후원 통합** |

> Pinduoduo 가 한국에서 못 한 "카카오 + 식당 O2O" 침투를 우리가 한다.

## 2. 셀러 기대효과 — 왜 유어딜에서 시작해야 하는가

### 2-1. 진입장벽 ZERO

- **카카오 로그인 → 5분 내 첫 방송**
- YouTube 채널만 있으면 즉시 셀러 등록 (별도 사업자 등록증 미요건, 통신판매업은 매출 기준 도달 시)
- OBS / 모바일 카메라 송출 — **브라우저만 있으면** 시작 (앱 설치 X)
- 자체 쇼핑몰 구축 비용 0원 (Shopify 월 $29 + 도메인 + 결제 PG 가입 부담 X)

> 비교: 자체 YouTube 채널 + Shopify 시작 시 평균 2~3개월 + 200만원 초기 비용.
> 유어딜은 **0원 / 5분**.

### 2-2. 다중 수익원 — 한 플랫폼에서 4가지 수익화

| 수익원 | 플랫폼 수수료 | 셀러 수령 |
|---|---|---|
| 상품 판매 | 10% | 90% |
| 라이브 후원 | 15% (기본) | 85% |
| 실시간 경매 | 10% | 90% |
| 그룹바이 (공구) | 5% | 95% |

> 단일 셀러가 **상품 + 후원 + 경매 + 공구** 4개 채널로 동시 매출. 다른 플랫폼은 한 가지에 특화돼 있다.

### 2-3. 셀러 등급 시스템 — 성장형 보상

```
신규 → 브론즈 → 실버 → 골드 → 플래티넘
```

- 등급 진급 시 **수수료 차감** (플래티넘은 최저 수수료)
- 알고리즘 **노출 가중치 상승** — 상위 등급 셀러가 홈/카테고리 상단
- 어드민이 셀러별로 `commission_rate` 개별 조정 가능 — 협상 여지

### 2-4. AI 매칭 — 영업 없이 협업

- **유어딜 에이전시 매칭 추천**: 카테고리/규모/지역 기준으로 적합한 MCN 자동 추천
- **셀러간 PK 배틀 매칭**: 비슷한 규모 셀러와 라이브 대결 → 양쪽 시청자 풀 합산

### 2-5. 무료 CRM 인프라

- **알림톡 무료 크레딧** (가입 시 지급)
- 카카오 공유 → 클릭 → 구매 **풀 퍼널 분석**
- 단골 고객 자동 태깅 + 리타겟팅 알림

> 별도 CRM SaaS (예: 채널톡 월 5만원, 카카오 알림톡 건당 9원) 비용 절감.

### 2-6. 업계 최단 정산

| 정산 항목 | 유어딜 | 업계 평균 |
|---|---|---|
| 상품 (배송 확정 후) | **D+5일** | D+15~30 |
| 라이브 후원 | **D+10일** | (지원 안 함) |
| 그룹바이 | **D+7일** | D+30 |

> 현금흐름 = 셀러 생존. **D+5** 는 업계 최단 클래스.

## 3. 에이전시 (MCN) 기대효과 — 왜 유어딜에서 셀러를 키우는가

### 3-1. 멤버 셀러 통합 대시보드

- 모든 소속 셀러의 **매출 / 방송 / 시청자 / 후원 / 정산** 한 화면
- 셀러별 KPI 비교 → 부진 셀러 코칭 우선순위 자동 추출
- 일/주/월 리포트 자동 생성

### 3-2. PK 배틀 — 자체 토너먼트 운영

- **소속 셀러 간 PK 매칭** → 시청자 트래픽 합산 + 응원 후원
- 토너먼트 시즌 운영 → 우승자 노출 부스트 + 상금
- 트래픽 폭증으로 **신인 셀러 데뷔 가속**

### 3-3. 프로모트 부스트 패키지

- 특정 멤버 셀러 **노출 강화 패키지** 구매
- 홈 / 카테고리 상단 / 푸시 알림 노출
- 신인 데뷔 / 시즌 캠페인 / 신상품 런칭에 활용

### 3-4. 그룹 인센티브

- 에이전시 전체 목표 매출 달성 시 **추가 수수료 환급**
- 멤버 셀러 동기부여 → 자연스러운 그로스 룹

### 3-5. AI 셀러 매칭 — 신규 영입 자동화

- 유어딜에 가입한 미소속 셀러 중 **에이전시 카테고리/규모 적합 셀러 추천**
- 영업 자동화 → MCN 인력 비용 절감

### 3-6. 정산 자동화

- 멤버 셀러 정산을 **에이전시가 일괄 관리**
- 셀러별 수수료 차등 설정 가능
- 세무 처리용 **CSV / 세금계산서 자동 발급**

## 4. 소비자 "wow" 모먼트 — 왜 쿠팡 대신 유어딜인가

### 4-1. 경매 낙찰의 짜릿함

> "라이브 방송 보다가 5만원 짜리 한정판이 3만원으로 떨어지는 순간 — 누가 먼저 클릭하느냐."

- 셀러가 라이브에서 경매 시작 버튼
- 가격이 실시간 하락 → 누군가 클릭하면 즉시 낙찰
- 즉시 결제 → 1주일 후 받기
- 채팅창에 "낙찰 축하합니다!" 표시

> 쇼핑이 게임이 된다.

### 4-2. 식사권 공구 — 친구 모아 30% 할인

> "강남 한우 5만원 → 50명 모이면 3만 5천원. 카카오톡으로 친구 5명 초대하면 추가 5천원 할인."

- 식사권 그룹바이 50명 달성 시 **자동 30% 할인 적용**
- 친구 카카오 공유 → 인당 추가 할인
- 식당 가서 **앱 화면 보여주기만** 하면 OK (종이 쿠폰/플라스틱 X)
- QR 스캔 → 사용 처리 자동

### 4-3. 셀러 후원 → 라이브 노출 → 직접 인사

> "좋아하는 셀러에게 1만 딜 후원 → 라이브 화면에 내 닉네임 큰 글씨 → 셀러가 마이크 잡고 '○○님 감사합니다!'"

- 트위치 도네이션과 동일한 **팬덤 메커니즘**
- 후원 메시지 라이브 자막
- 단골 후원자 자동 VIP 태깅 → 신상품 우선 안내

### 4-4. 식사권 디지털 사용 — 종이 쿠폰 시대 종료

- 구매 즉시 앱에 **디지털 식사권** 발급
- 식당 가서 앱 보여주기 → 점원이 "사용 완료" 버튼
- 사용 가능 매장 지도 (`/restaurant-map`) 에서 한눈에 확인
- 쓰지 못하면 **자동 환불**

### 4-5. 카카오 네이티브 경험

- 카카오 로그인 1초
- 카카오페이 결제 1초
- 친구한테 카카오톡 공유 1초
- 알림톡으로 배송 / 사용 안내

> 한국 사람이 가장 익숙한 동선 그대로.

## 5. 핵심 기능 한눈에

### 5-1. 라이브 방송

- WebRTC + HLS 듀얼 스트리밍
- 9:16 풀스크린 모바일 우선, PC 도 430px 액자 모드
- 실시간 채팅 / 후원 / 경매 / 좋아요
- 송출: 브라우저 카메라 + OBS RTMP 둘 다 지원

### 5-2. 상품 / 주문

- 상품 등록 → 옵션 / 재고 / 배송비 → 라이브에 즉시 노출
- 주문 → 토스 결제 / 가상계좌 / 카카오페이 / 딜 포인트
- 배송 추적 → CJ대한통운 API 연동
- 자동 배송 확정 → 정산 D+5

### 5-3. 그룹바이 (공동구매)

- 목표 인원 달성 시 자동 할인 적용
- 미달성 시 **전액 자동 환불**
- 식사권 카테고리 특화

### 5-4. 라이브 경매

- 셀러가 시작가 / 최저가 / 시간 설정
- 가격 자동 하락 (Dutch auction)
- 첫 클릭한 사용자 낙찰
- 즉시 결제 연동

### 5-5. 후원 (도네이션)

- 최소 500딜 (=500원)
- 라이브 화면 자막 + 채팅 강조
- 셀러 정산 D+10
- Turnstile 봇 차단

### 5-6. 딜 포인트

- 1원 = 1딜 (수수료 ZERO 충전)
- 후원 / 상품 결제 / 경매 모두 사용
- 최소 후원 500딜

### 5-7. 셀러 등급 / 수수료

- 신규 → 브론즈 → 실버 → 골드 → 플래티넘
- 진급 시 수수료 차감 + 노출 가중
- 어드민 개별 조정 가능

### 5-8. 식당 O2O

- 디지털 식사권 발급 / 사용
- 매장 지도 (`/restaurant-map`)
- 사용 안 함 → 자동 환불
- QR 없이 앱 화면만으로 처리

### 5-9. PK 배틀

- 셀러간 / 에이전시간 라이브 대결
- 시청자 후원 합산으로 승부
- 자체 토너먼트 시즌 운영

### 5-10. AI 추천

- 셀러 ↔ 에이전시 매칭
- 셀러 ↔ 셀러 PK 매칭
- 사용자 ↔ 라이브 추천 피드

## 6. 기술 신뢰성

| 항목 | 내용 |
|---|---|
| 인프라 | Cloudflare Pages + Workers (글로벌 엣지, 99.99% SLA) |
| DB | Cloudflare D1 (분산 SQLite, 자동 백업) |
| 결제 | 토스페이먼츠 + 카카오페이 + 딜 포인트 |
| 인증 | 카카오 OAuth (KR), Firebase Auth (Global) |
| 봇 차단 | Cloudflare Turnstile |
| Rate Limit | KV 기반 IP/계정 limiter |
| 다국어 | 한/영/일/중/스/프 6개 언어 |
