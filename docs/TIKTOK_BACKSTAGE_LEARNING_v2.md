## TikTok 자료 핵심 인사이트 (23장 정리)

### 1. 다층 영입 시스템
- QR 코드/링크 (7일 유효)
- IM 인스턴트 메시지 (14일 유효)
- 메시지 템플릿 (변수 치환)
- 짧은 동영상 모집
- 온라인/오프라인 채널 다양화
- 흔한 오해 명시 (모집 가이드)

### 2. 에이전시 등급
- TikTok: 신규(6m 이내) / 주니어(6m+) / 시니어(전월 500만+)
- **우리 적응**: 브론즈 (가입 후 90일 또는 누적 매출 적음) / 실버 (90일+ 또는 누적) / 골드 (월 매출 일정 수준)

### 3. 6대 KPI (참고용)
TikTok 의 6개 → 우리 적응:
1. ~~다이아몬드~~ → **총 딜 매출**
2. **라이브 진행률** (라이브한 셀러 / 전체 셀러)
3. **유효 라이브 진행률** (30분+ 라이브한 셀러 비율)
4. **활성 셀러 수** (이번 주 1+ 라이브)
5. **유효 활성 셀러 수** (이번 주 30분+ 라이브)
6. **신규 셀러 수** (이번 주 가입)

### 4. 시청자/매출 부스팅
- 이벤트 3종: Self / TikTok 공식 / Top 에이전시 공동주최
- 시청자 4단계 심리 (선물 보내는 이유) → 4단계 세그먼트
- PK 이벤트 (셀러 vs 셀러 매출 경쟁)

### 5. 셀러 가이드
- 라이브 환경 체크리스트 (배경/조명/마이크)
- 자기소개 10~20초 가이드
- 시청자 응대 매뉴얼

### 6. 운영 자동화
- 송장 자동 생성 (이미 M6 구현)
- 자동 정산 (이미 P0#3 구현)
- AI 모더레이션
- FAQ 봇

## Phase 1 Quick Win (1~3주)

### 1. 에이전시 등급 시스템 (브론즈/실버/골드) — 6h
- DB: `agencies.tier` 컬럼 추가 (`bronze` | `silver` | `gold`)
- Cron: 월 1회 자동 평가 (강제 X, 참고용)
- 정책 (느슨):
  - **브론즈**: 가입 후 60일 이내 OR 전월 매출 100만 딜 이하
  - **실버**: 가입 60일+ AND 전월 매출 100만~500만
  - **골드**: 전월 매출 500만+ AND 활성 셀러 5명+
- 어드민이 수동 변경 가능 (locked 플래그)

### 2. 부진 셀러 자동 알림 — 4h
- Cron 매일: 7일 무라이브 / 30일 무매출 셀러 검출
- `agency_notifications` 테이블 INSERT (이미 0222 마이그레이션 있음)
- 에이전시 대시보드에 표시

### 3. QR/링크 영입 시스템 — 6h
- DB: `agency_invite_codes` (code, agency_id, expires_at, used_count)
- 라우터:
  - POST `/api/agency/invites` — 코드 발급
  - GET `/api/agency/invites` — 목록/통계
  - GET `/api/invite/:code` — 공개 (가입 폼용)
- 셀러 register 라우트: `?invite=<code>` 받으면 자동 에이전시 매핑
- 페이지:
  - `/agency/invites` — 발급/관리
  - `/seller/register?invite=X` — 에이전시 표시 + 자동 매핑

### 4. 에이전시 6대 KPI 대시보드 — 8h
- 기존 `/agency` 메인에 카드 6개 추가 (이미 있는 위젯 옆)
- 모두 **참고용** — 강제 패널티 없음
- 주간/월간 토글
- 등급별 평균 비교 (벤치마크)

### 5. 셀러 부트캠프 7일 온보딩 — 12h
- DB: `seller_onboarding_progress` (seller_id, step_key, completed_at)
- 7단계:
  1. 프로필 완성 (사진/소개/주소)
  2. 첫 상품 등록
  3. 첫 라이브 (15분 이상)
  4. 첫 후원 받기
  5. 첫 결제 받기
  6. 알림톡 첫 발송
  7. 7일 완주
- 셀러 대시보드: 진행률 위젯 (X/7 단계)
- 7일 완주 시 자동 보상 (딜 1만 지급)
- 에이전시 대시보드: 부트캠프 진행 셀러 모니터링

### 6. 셀러 라이브 시작 가이드 모달 — 4h
- 셀러 라이브 페이지에서 [라이브 시작] 클릭 시 첫 회 모달
- 체크리스트:
  - 자기소개 10~20초 준비
  - 배경/조명/마이크 확인
  - 라이브 제목 매력적으로
  - 첫 시청자 응대 멘트
- "다시 보지 않기" 옵션

### 7. 에이전시 공개 브랜딩 페이지 — 8h
- 공개 라우트: `/agency/<slug>` (현재 어드민 전용 → 공개 페이지 신설)
- 표시: 에이전시명/소개/소속 셀러 수/누적 매출/대표 셀러
- "에이전시 가입 신청" 폼 (= QR 영입과 결합)
- 에이전시가 직접 편집 가능

**Phase 1 총합**: ~48시간, 7개 기능
