# [경쟁사 분석] 셀파이 (sellpie.co.kr, ㈜에디톨로지) — 2026-06-10

> 사용자 제보: 연 매출 약 50억. 공개 데이터: 2021.7 베타 시작 → 2022년 매출 31억, 온라인 판매기업(셀러) 2천여 개 유치.
> 사이트는 봇 차단(403) — 검색 인덱스/언론 기사/확보한 공급계약서(`competitor-sellpie-supply-contract.md`) 교차 분석.

## 1. 정체 — 한 줄 요약
**"스마트디지털기기 한 카테고리만 파는, 풀필먼트(자체 물류) 기반 B2B 위탁판매 도매 플랫폼."**
구조는 우리 유통스타트와 동일(제조사 ↔ 플랫폼 ↔ 온라인 셀러)이나, **물류를 직접 잡고 있다**는 점이 결정적 차이.

## 2. 수익 모델 해부 — 50억이 나오는 구조
- **위탁 도매 매출 전체가 회사 매출로 인식**: 셀러가 주문 넣으면 셀파이가 공급가에 마진을 얹어 청구 → 거래액 자체가 매출. (수수료 모델이 아님 — 매출 50억 ≒ 연 거래액 50억, 영업이익은 별개)
- **무재고 셀러 대량 모집**: 셀러는 재고/물류 부담 0 (위탁) → 진입장벽 0 → 2천 셀러. 셀러당 연 거래 ~250만원만 돼도 50억.
- **마진 스프레드**: 셀러 마진 "평균 30%, 최대 40%" 보장 — 제조사 공급가와 셀러 공급가 사이가 플랫폼 몫.
- **묶음배송 + 당일배송**: 자체 물류센터(김포 240평 '셀카고' 자회사)라 가능 — 셀러가 여러 제조사 상품을 한 박스로 → 객단가↑ + 타 도매 대비 차별화.
- **최저가 정책 + 신고 시스템**: 계약(4조·11조)으로 가격 통제 → 셀러 간 출혈경쟁 차단 → 마진 30% 약속이 지켜짐 → 셀러 잔존율↑. **가격 신뢰가 핵심 자산.**

## 5. 액션 인사이트 (우선순위)
1. **① 포지셔닝 — "전체 도매몰"이 아니라 "카테고리 1위 도매몰"로**: 셀파이의 진짜 비결은 풀필먼트보다 "스마트디지털기기 국내유일 B2B"라는 니치 독점 카피. 우리는 멀티몰 인프라가 있으니 **첫 몰 하나를 한 카테고리(예: 식품)로 좁혀 '국내유일 ○○ B2B'를 선점**하고, 검증되면 몰 복제. 전 카테고리 백화점식은 신뢰도 못 쌓음.
2. **② 셀러 모집 기수제 (비용 0, 즉시)**: "유통스타트 1기 프리미엄 셀러 모집 (○명 한정)" — 가입은 같지만 마감일+혜택(상위 등급 6개월 보장 등)으로 포장. 셀파이가 5기까지 우려먹는 검증된 퍼널.
3. **③ 계약서 체결 플로우**: 최저가 신고 기능을 만들었으니 제재의 계약 근거가 필요 — 가입 승인 시 온라인 동의(또는 모두싸인류). 계약서 초안은 archive 한 셀파이 계약서 구조 참고.
4. **④ 묶음배송은 단계적으로**: 자체 물류는 자본 게임이라 당장 X. 중간 단계 = 같은 제조사 상품 묶음배송(이미 가능) → 3PL 제휴 묶음(중기) → 자체 풀필먼트(장기, 거래액 검증 후).
5. **⑤ 마진 수치를 전면에**: 셀파이는 "평균마진 30%! 최대 40%"를 모든 채널에 박음. 우리도 카탈로그/배너/모집 카피에 구체 숫자 고정 (우리는 이미 '판매마진 40%' 메뉴 있음 — 가입 랜딩/배너에도).

## 6. 리스크/한계 (그들의 약점 = 우리 기회)
- 단일 카테고리 = 확장 천장. 우리 멀티몰은 카테고리별 복제 가능.
- 자체 물류 = 고정비 부담 (240평 임차/인력) — 시장 침체 시 역레버리지.
- 셀러 2천 중 활성 비율 불명 — 기수제 모집을 계속 하는 것 자체가 이탈이 있다는 신호.
