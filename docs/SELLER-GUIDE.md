# 셀러 가이드

## 셀러 회원가입 / 로그인

### 회원가입
1. `/seller/register` 페이지에서 가입
2. 필수 정보: 아이디, 비밀번호, 이름, 이메일, 사업자명
3. 가입 후 어드민 승인 대기 (status: `pending` -> `approved`)

### 로그인
1. `/seller/login` 페이지에서 아이디/비밀번호 입력
2. JWT 토큰 발급 -> `seller_token`으로 localStorage에 저장
3. 모든 API 요청에 `Authorization: Bearer <seller_token>` 자동 첨부

## 라이브 방송 방법

### 사전 준비
1. YouTube 채널 보유 (라이브 스트리밍 활성화 필요)
2. 방송 소프트웨어: **프리즘 라이브** (모바일) 또는 **OBS Studio** (PC)

### 방송 시작 순서

1. **YouTube OAuth 연동**
   - 셀러 대시보드 -> 라이브 방송 -> YouTube 계정 연동
   - Google OAuth 인증 완료

2. **방송 생성**
   - 라이브 방송 페이지에서 "새 방송" 생성
   - 제목, 설명, 예약 시간 입력
   - 상품 연결 (방송 중 판매할 상품 선택)

3. **스트림 키 복사**
   - 방송 생성 후 표시되는 **스트림 키**(Stream Key) 복사

4. **방송 소프트웨어 설정**
   - 프리즘/OBS에서 스트림 키 붙여넣기
   - 서버 URL: YouTube RTMP 서버
   - 방송 시작 버튼 클릭

5. **방송 중 관리**
   - 실시간 채팅 확인
   - 현재 판매 상품 변경
   - 후원 메시지 확인

## 상품 등록

### 개별 등록
1. 상품 관리 -> "새 상품" 버튼
2. 입력 항목:
   - 상품명, 가격, 원가, 할인율
   - 대표 이미지, 상세 이미지
   - 재고 수량 (`stock` 필드)
   - 카테고리
   - 옵션 (색상, 사이즈 등)
3. 저장 시 `is_active = 1` (활성 상태)

### CSV 대량 등록
1. 상품 관리 -> "대량 등록" 버튼
2. CSV 템플릿 다운로드
3. 엑셀/구글시트에서 상품 정보 입력
4. CSV 파일 업로드 (`/api/bulk-upload/*`)
5. 미리보기 확인 후 등록 완료

## 주문 처리

### 주문 상태 흐름

```
DONE (결제 완료) -> PREPARING (상품 준비) -> SHIPPING (배송 중) -> DELIVERED (배송 완료)
```

### 주문 처리 순서
1. **주문 확인**: 새 주문 목록에서 주문 확인
2. **상태 변경**: DONE -> PREPARING (상품 준비 시작)
3. **운송장 등록**: 택배사 선택, 운송장 번호 입력
4. **배송 시작**: PREPARING -> SHIPPING (운송장 등록 시 자동 또는 수동)
5. **배송 완료**: SHIPPING -> DELIVERED

### 운송장 등록
- 택배사(`courier`): CJ대한통운, 한진택배, 롯데택배 등
- 운송장 번호(`tracking_number`) 입력
- 등록 시 주문자에게 알림 발송

## 정산

### 정산 구조
- **수수료율**: 셀러별 commission_rate (기본 10%)
- **정산 금액**: 주문 총액 - 수수료
- **정산 상태**: `pending` -> `settled`

### 정산 신청
1. 정산 페이지 (`/seller/settlements`)에서 정산 가능 금액 확인
2. 정산 신청 버튼 클릭
3. 어드민 승인 후 정산 처리
4. 등록된 은행 계좌로 입금

---

## 바코드 재고 관리

### 바코드 생성
1. 재고 관리 페이지 (`/seller/inventory`)
2. 상품별 바코드 자동 생성
3. 바코드 라벨 인쇄

### 카메라 스캔
1. 모바일에서 카메라 스캔 기능 사용
2. 바코드 스캔 -> 상품 자동 인식

### 입출고 관리
- **입고**: 바코드 스캔 후 입고 수량 입력 -> `stock` 증가
- **출고**: 바코드 스캔 후 출고 수량 입력 -> `stock` 감소
- 입출고 이력 자동 기록

## 브랜드메시지 (알림톡)

### 크레딧 충전
1. 브랜드메시지 페이지 (`/seller/alimtalk`)
2. 크레딧 패키지 선택 후 충전 (결제)
3. 충전된 크레딧으로 메시지 발송

### 메시지 발송
1. 발송 대상 선택 (주문 고객 등)
2. 메시지 템플릿 작성
3. 발송 (크레딧 차감)
4. 발송 제한: 분당 60건 (rate limit)
