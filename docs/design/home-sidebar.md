# 홈 사이드바 + 메인 디자인 시안

**시안 받은 날**: 2026-05-06
**상태**: ✅ 완료 (2026-05-06)

## 시안 개요

PC 화면 좌측에 고정 사이드바, 중앙에 FLASH DEALS 히어로 + 타임딜 그리드.

### 좌측 사이드바 (고정폭, 약 220px)

**상단 로고**: `UR DEAL` 워드마크

**MENU 섹션**:
- 🏠 홈
- ▶ 라이브
- 🔍 둘러보기
- 공동구매
- 🎫 식사권

**CATEGORY 섹션**:
- 치킨·피자
- 한식·분식
- 카페·디저트
- 일식·돈까스
- 아시안·양식
- 뷰티·헬스
- 리빙·인테리어

**MY 섹션**:
- 마이페이지
- 주문내역
- 찜
- 내 식사권

**좌하단 CTA 박스** (분홍 배경):
- "유어딜 앱으로"
- "라이브 알림 + 이지바 추천 / 앱 전용 혜택 받을 수 있어요"
- 검정 버튼: "앱 다운로드"

### 상단 탑바

- 좌: `UR DEAL` 로고 (실제는 사이드바와 통합)
- 가운데: 검색바 (placeholder: "라이브, 식사권, 상품 검색")
- 우측: 🔴 LIVE 배지 / 판매자센터 / 고객센터 / 🛒 장바구니 / 분홍 "로그인" 버튼

### 중앙 메인 컨텐츠

**FLASH DEALS 히어로** (분홍 그라데이션 박스):
- 🔥 FLASH DEALS · 매일 자정·정오 6시 갱신
- "오늘 자정까지 14:23:08 남았어요" (큰 카운트다운)
- "지금 진행 중인 타임딜 6개 · 평균 할인율 47%"

**진행 중인 타임딜 그리드** (3열 × 2행 = 6개):
- 카드: 좌상단 빨간 시간 배지 (남은 시간) / 우상단 검정 할인율 배지
- 본문: 상품 이미지 (분홍 placeholder) / 상품명 / 현재가 (큰 글씨) + 정가 (취소선)
- 하단: 판매 진행률 바 + "87% · 87/100개" 텍스트

## 현재 구현 vs 시안 차이

| 항목 | 시안 | 현재 (`DesktopLiveSidebar.tsx`) |
|---|---|---|
| 섹션 구분 | MENU / CATEGORY / MY 3개 | 단일 nav |
| MENU 항목 | 홈, 라이브, 둘러보기, 공동구매, 식사권 | 추천/탐색/팔로잉/라이브/쇼핑/검색/프로필 |
| CATEGORY | 7개 음식 카테고리 | 없음 |
| MY 항목 | 마이페이지/주문내역/찜/내식사권 | 프로필 1개만 |
| 앱 다운로드 CTA | 좌하단 박스 | 없음 |
| 표시 범위 | 모든 페이지 (HIDE 제외) | 라이브/쇼츠만 (xl: 1280px+) |
| 메인 히어로 | FLASH DEALS 카운트다운 | (아직 다른 디자인) |
| 타임딜 그리드 | 6개, 시간/할인 배지, 진행률 | (아직 다른 디자인) |
