## 대화에서 사용자가 최종 안착한 결정 (중요)
- 팔로우 기능 **없음**, 실시간 구매 티커 **없음**, 팔로워/누적절약/핀 통계 섹션 **없음**.
- 프로필 섹션: **가로형 컴팩트**(아바타 좌 · 이름/핸들/소개 우), **그라데이션 색 제거 → 흰 배경 + 얇은 구분선**, 인증 체크 배지.
- **SNS 버튼**(유튜브 레드 / 인스타 그라데이션 / 틱톡 다크) — 프로필 소개 아래, 새 탭 이동. 본인은 편집 가능. ⚠️ 신규 데이터(유저 SNS 필드) 필요.
- **검색창**(탭과 함께 sticky) — 이름 + 추천코멘트(note) 라이브 필터 + ✕ + 빈결과.
- **카드 (에디토리얼, AI 느낌 제거)**: 좌상단 **번호 코너플래그**(coral) · 이미지 위 **다크 가격칩**(할인%·정가취소선·판매가) · **절약액 초록 텍스트** · **좌측 coral rule + 인용문 note**(이모지 제거) · 출처는 이름만("큐레이터" 명칭 제거). 조회·저장 통계 제거. 마감 D-N pill은 카테고리 줄 우측 상단.
- **#1 = 풀폭 히어로 카드**(강력추천), 2~5 = 2열 그리드.
- 본인 뷰: 이번주 적립 strip · 방문자 미리보기 카드 · 인라인 편집 · 핀 드래그 정렬 · "추천 코멘트 없음 ⚠" 유도 · 콘솔 버튼.
- 그리드 끝 **"+ 핀 추가하기" 점선 카드**(본인 한정).
- 빈 상태: **흐릿한 샘플 ghost 핀(그라데이션 마스크로 아래로 페이드)** + 떠있는 CTA "첫 핀을 추가해 보세요"(A안 채택, "예시 미리보기" 워터마크 제거). B안(샘플 없음)도 비교용.
- 라인 SVG 아이콘(이모지 제거), 하단탭바.
