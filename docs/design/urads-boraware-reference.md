# 유어애즈(UR Ads) — 보라웨어(Boraware) 서비스 레퍼런스

> 대표가 2026-06-27 채팅으로 보라웨어 각 서비스 랜딩페이지 스크린샷을 순차 공유.
> 유어애즈는 보라웨어식 **종합 검색광고 마케팅 솔루션**을 목표로 함. 이 문서는 각 서비스의
> 기능/카피/필요한 네이버 API/구현 의존성을 박제(채팅 이미지는 세션 종료 시 사라짐 — CLAUDE.md 룰).

**공통 랜딩 템플릿**(모든 서비스 동일): 상단 가입혜택 3컬럼 → 히어로(서비스 한줄정의 + 노트북 목업) →
Cheil 추천 바 → 기능 6블록(좌/우 교차, 각 스크린샷 목업) → 업계 전문가 추천(Openinsight·Cheil·동아일보·Microsoft) →
"1급 검색광고 고미에디터가 도와드립니다" 배너 → **플랜별 비용안내**(체크리스트 카드) → "900개+ 기업 함께"(900억원·1589만개·76만건·97.3%) →
광고주 후기(사진+인용) → 수상/인증 로고 → 하단 CTA("최대 14일 무료체험").

브랜드: 유어애즈 / UR Ads. 서비스 표면 `/ads`, API `/api/ads`, 레이아웃 `MarketingLayout`(전체 PC폭, 도매몰 분리 패턴).

## 1) 자동입찰 (Auto-bidding) — "원하는 순위, 원하는 CPC로! 네이버 자동입찰 솔루션"

부제: "광고를 93.3% 검색사용자, 원하는 순위 노출 + CPC 최대 15% 절감".

**기능 6블록**:
1. **목표순위·최저입찰가만 정하세요** — 최저가/목표순위 노출, CPC 최대 15% 절감.
2. **자동입찰 핵심은 순위정확도** — 원하는 CPC·원하는 순위 노출정확도 평균 약 3% 확인.
3. **(소형입찰 기능)** — 상호별 고객 쇼핑키워드 제안 + 편리한 제외키워드 관리.
4. **시간대별 맞춤 입찰전략** — 피크시간 전략 / 마감 맞춤전략 / 주말 전략 등.
5. **세부 지역입찰 OK · 모바일입찰 OK** — 원하는 지역·단말기에 정확히 노출.
6. **대량 입찰설정** — 기존성과 참조 입찰설정 + CSV 입찰설정 등.

**가격(플랜 4종)**: 자동입찰 스타트 / 일반 / 부스터 / (프리미엄) — 키워드 수·기능 차등.

**네이버 API 매핑 & 의존성**:
- 입찰가 조회/변경: 검색광고 API **Adgroup/Keyword bidAmt** (PATCH). ⚠️ **고객사 광고계정이 우리 관리계정(47982)에 연동돼야** write 가능.
- 목표순위→입찰가 추정: **Estimate API**(`/estimate/average-position-bid`, `/estimate/exposure-minimum-bid`) — 목표순위·최소노출가 추정.
- 실제 평균노출순위: **StatReport**(평균노출순위 지표). ⚠️ **순위측정은 공식 API only — SERP 스크래핑 금지**(2026-04-22 제거, PIPA/정보통신망법).
- 엔진: Cloudflare **Cron**(주기적 폴링 → 추정/조정 → bidAmt PATCH). 고정IP 불필요(검색광고 API HMAC 서명).
- 시간대/지역/모바일 입찰: 검색광고 API의 입찰가중치(bidweight)·스케줄 — API 지원범위 라이브 검증 필요.

## 2) 부정클릭 (Click-fraud prevention) — "국내유일 100% 자동관리! 검색광고 부정클릭 방지시스템"

부제: "부정클릭/무효클릭으로부터 스마트한 광고비를 지켜드립니다".

**기능 6블록**:
1. **단 한 번 설정으로 부정클릭 100% 자동관리** — 편리함.
2. **악성 부정클릭(IP 바꿔 접속) 완벽 차단** — 광고비 낭비 주범 차단.
3. **(선택옵션) 무효클릭 접속자 광고 차단** — 출처·유도부터 별도조치 광고제어.
4. **접속자 위치정보 제공** — 부정클릭 집중 지역 확인.
5. **접속자 활동성 분석** — 단골고객 vs 부정접속자 구분.
6. **PC·모바일앱 자동보고** — IP까지 꼼꼼하게 보고.

**가격**: 부정클릭 방지시스템(₩12,460원/1만건당 류) + 일반/세타.

**네이버 API 매핑 & 의존성 (⚠️ 아키텍처 다름)**:
- 클릭/방문자 수집: **광고주 랜딩페이지에 추적 스크립트(픽셀)** 삽입 → 방문 IP·UA·행동 로그(우리 D1/Queue). 자체 인프라 필요(네이버 API 아님).
- 부정 IP 차단: 네이버 검색광고 **"노출제한 IP"**(계정당 최대 600개) 에 등록. ⚠️ **검색광고 공식 API 가 노출제한 IP 관리를 노출하는지 라이브 검증 필요** — 미지원이면 반자동(권고 리스트 + 수동등록 안내).
- ⚠️ **개인정보(PIPA)**: 방문자 IP/위치 수집 → 처리방침 고지 + 목적제한. legal 검토 필수(2026-04-22 스크래퍼 제거와 같은 리스크 영역).

## 3) 키워드확장 (Keyword expansion) — "고성과 키워드를 무한확장! 지능형 키워드 자동확장"

부제: "연관검색어 + 자동완성 + 키워드도구 등을 한번에 확장하기".

**기능 5블록**:
1. **(국내유일) 지능형 고성과키워드 자동확장** — 매출 높고 성과 좋은 키워드 자동확장.
2. **연관검색어 + 자동완성 + 키워드도구** — 숙련 마케터 방식으로 확장(NAVER 검색창 목업).
3. **수집 키워드 원클릭 자동등록** — 수집과 등록을 한번에.
4. **조건 지정 → 정기 자동확장** — 원하는 시기에 자동, 항상 최신 키워드 유지.
5. **지정 키워드 확장** — 신규 캠페인/아이템을 핵심키워드 몇 개로 확장.

**가격**: 키워드 자동확장(₩12,460원/1만건당 류) + 할인/세타.

**네이버 API 매핑 & 의존성**:
- 연관키워드 발굴: 검색광고 **RelKwdStat**(연관키워드+월검색량) — ✅ **이미 구현됨**(2026-06-27, `searchad-client.ts` `relatedKeywords`).
- 자동완성: 네이버 **자동완성**(ac.search.naver.com) — read, 키 불필요. 추가 가능.
- 고성과 키워드 선별: **StatReport**(전환/매출 기준 정렬) → 상위 키워드 seed.
- 원클릭 등록: 검색광고 API **Keyword create**(ad group 에 키워드 추가). ⚠️ **고객사 광고계정 연동 필요**.
- 정기확장: Cloudflare **Cron**.

## 4) 통합실적 (Unified performance) — "모든 매체성과를 한눈에! 다매체 통합실적 자동분석"

부제: "네이버·구글·카카오까지, 타겟게이즈·모바일 실적을 한번에".

**기능 3블록**:
1. **다매체 광고실적 자동수집·자동분석** — 여기저기 흩어진 실적, 한눈에. 네이버/구글/카카오 등 주요 매체 성과를 하나의 보고서로.
2. **매일·매주·매월 자동보고 시스템** — 바빠도 보고는 꼭. 정해진 주기마다 자동분석 리포트 발송.
3. **보라웨어 통합모니터링** — 광고내역·입찰관리·부정클릭·키워드확장 등 자동화 솔루션 운영상황을 한 화면에.

**가격**: 통합실적 자동분석(₩12,600원/1만건당 류) + 할인/혜택.

**네이버 API 매핑 & 의존성**:
- 네이버 실적: 검색광고 **StatReport / MasterReport**(캠페인·키워드·디바이스·시간대 지표). read — 광고계정 연동 후.
- 멀티매체(구글/카카오): Google Ads API / Kakao Moment API — 후순위(네이버 우선).
- 자동보고: Cloudflare **Cron** + 이메일(Resend, 이미 보유) / 알림톡(Aligo, 보유).
- **통합모니터링** = 유어애즈 자체 솔루션(입찰·키워드·부정클릭) 운영로그를 D1 에 모아 한 대시보드 — 우리 인프라.

## 5) AI마케터 — "검색광고 ROAS 올리는 가장 쉬운 방법, AI마케터"

부제: "전략부터 운영까지, AI마케터로 광고성과(ROAS·CPA) 6주 향상".

**기능 4블록**:
1. **(국내유일) AI마케터 자율운영** — 자료수집·전략설정·광고운영까지 100% 자율(연관키워드+실적 데이터로 AI 운영).
2. **ROAS·CPA 자동관리** — 8주 안에 광고성과 최소 30% 이상 향상.
3. **프리미엄 무료대행 파트너 전용** — 입찰·키워드·부정클릭·통합실적 자동운영과 함께, 평생무료.
4. **네이버 파워링크 전환수 300개만 있으면 즉시 사용** — 최근 1년 내 누계 전환 ≥300(AI 학습 최소데이터).

**사용안내**: STEP1 파워링크 최소 300전환 확인 → STEP2 상담신청 → STEP3 회원가입·전환서비스.

**네이버 API 매핑 & 의존성 (⭐ 우리 강점)**:
- 입력 데이터: **StatReport**(전환/ROAS/CPA) + **RelKwdStat**(연관키워드 ✅보유) + **Estimate**(입찰추정).
- 두뇌: **Claude(Opus/Sonnet)** — 이미 보유. 데이터→진단→실행계획(예산배분·키워드 추가/제외·입찰조정·소재 제안).
- 실행: 검색광고 API write(bidAmt·keyword·budget) — 광고계정 연동 후. read-only 진단/추천은 **연동 0개여도 먼저 가능**.
- → **유어애즈가 보라웨어 대비 구조적 우위**: 자체 LLM 보유로 "AI마케터"가 외부 의존 없이 동작.

## 6) 추가 네이버 API 서비스 아이디어 (대표 "더 찾아서 개발")

검색광고 API + 오픈API + 데이터랩 + 커머스API 로 가능한 확장 서비스. **읽기 위주(연동불필요)부터 빠르게**:

| 아이디어 | 필요 API | 연동필요? | 메모 |
|---|---|---|---|
| **순위/노출 추적** (공식) | 검색광고 StatReport(평균노출순위 추이) | 쓰기X/읽기 | ⚠️ SERP 스크래핑 금지 — 공식 지표만 |
| **경쟁사/쇼핑 최저가 모니터링·가격알림** | 오픈API 쇼핑검색 | 불필요 | ✅ **구현(2026-06-27)** — 워치 등록·일일 추적·내최저가 비교 |
| **데이터랩 트렌드 리포트** | 데이터랩 검색어트렌드(✅보유)·쇼핑인사이트 | 불필요 | 연령/성별/지역/기기 세분화 |
| **상품 소싱 추천**(뜨는 카테고리) | 데이터랩 쇼핑인사이트 카테고리 트렌드 | 불필요 | 도매몰과 시너지(뭘 떼올지) |
| **자동완성 키워드 발굴** | ac.search.naver.com 자동완성 | 불필요 | 키워드확장 보강 |
| **블로그/카페/뉴스 평판 모니터링** | 오픈API 검색(블로그/카페/뉴스) | 불필요 | 브랜드 언급 추적·알림 |
| **광고 소재 A/B 관리** | 검색광고 Ad(소재) API | 필요 | 제목/설명 변형 관리 |
| **예산/일예산 자동 페이싱** | 검색광고 Budget API | 필요 | 소진 과속/과소 방지 |
| **플레이스(지역업체) 모니터링** | 지역 검색 API | 불필요 | 지역 비즈 순위/리뷰 |

> 우선순위 제안: **읽기 위주(가격모니터링·트렌드리포트·평판모니터링)**를 광고계정 연동 없이 먼저 출시 →
> 사용자 확보 후 **쓰기(자동입찰·키워드등록·소재·예산)**를 광고계정 연동과 함께.

## 🔑 공통 핵심 의존성 (한 줄 요약)

대부분 기능의 **단 하나의 블로커 = "고객사 네이버 광고계정을 우리 관리계정(47982)에 연동 + write 권한"**.
- 연동 방식: 검색광고 **ManagedCustomerLink**(관리/대행 계정이 광고주 계정을 관리) — 멀티테넌트 허브.
- **읽기 전용**(연관키워드·실적조회)은 연동 0개여도 customer-level 로 동작 → **먼저 구현 가능**.
- **쓰기**(입찰변경·키워드등록·IP차단)는 광고계정 연동 후 → 그 다음 단계.
- 고정IP 불필요(검색광고 API). 발주수집(커머스API)만 고정IP 필요 → 별개 보류.
