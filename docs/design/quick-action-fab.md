# Quick Action FAB — 당근 패턴 확장형 플로팅 버튼

## 시안 출처
- 2026-05-24 사용자 공유 (당근마켓 캡처 2장)
- "유저가 바로 공동구매를 오픈하거나 하는 버튼"

## 시안 설명

당근마켓의 우하단 주황색 FAB 를 모델로:

**Closed state**: 우하단 둥근 컬러 버튼 (+ 아이콘 + 짧은 라벨)
- 당근: `+ 글쓰기` (주황)
- 우리: `+ 공구 열기` (핑크?)

**Open state (tap)**: 위로 stack 메뉴 펼침
- 당근: 알바/부동산/중고차/동네생활/모임/스토리 + (구분선) + 여러물건팔기/내물건팔기
- 우리: 역할별 분기 (아래 todo 참조)

**Close**: 같은 자리에 X 버튼

## 현재 vs 시안 차이

| 항목 | 현재 (KakaoConsultButton) | 시안 (Quick Action FAB) |
|---|---|---|
| 컴포넌트 | 단일 노란 카카오 말풍선 | 확장형 FAB + 메뉴 |
| 액션 | 카카오 상담 1개 | 다중 (공구 열기 / 라이브 / 상품 / 상담 등) |
| 색상 | 노랑 (#FFE812) | 핑크/주황 (브랜드) |
| 역할 분기 | 없음 (모두 동일) | 셀러 vs 일반 유저 다른 메뉴 |
| 상담 채널 | primary | 메뉴 한 항목으로 보존 |

### 비셀러 클릭 처리 — 후보 (사용자 결정 대기 중)
- (a) 셀러 가입 안내 모달 노출 ("공구를 열려면 셀러 가입이 필요해요" + CTA)
- (b) 셀러 가입 페이지로 즉시 이동
- (c) 메뉴에서 '공구 열기' 항목 자체 숨김 (셀러만 보임)
- (d) 일반 유저용 다른 액션으로 대체 ("공구 참여하기" / "내 주변 공구")
