# 링크샵 편집 모드 정리 (C) — 방문자 공개뷰에 가깝게

## 배경
사용자 신고: 링크샵(`/u/:handle`)이 "조금 밋밋하다". 진단 결과 두 원인:
1. **편집 모드 chrome 과다** — 헤더 아래로 관리 블록이 4개 세로로 쌓여 대시보드처럼 보임.
2. (별개) 테마/블록 커스터마이즈 부재 (= 향후 A/B 단계).

사용자 결정: **C(편집 모드 정리)부터.** 빠르게 체감, 저위험.

## 목표
오너가 보는 기본 화면 ≈ **방문자 공개뷰(헤더 프로필 + 핀 그리드)**.
관리 도구는 **한 줄 슬림 툴바**로 통합, 적립은 **한 줄 compact**.
편집 기능(인라인 이름/소개/사진/핀 삭제/순서/미리보기)은 **모두 보존** — 노이즈만 제거.

## 변경(After)
- **슬림 오너 툴바 1줄** (핀 ≥ 1, 미리보기/순서 모드 아닐 때):
  `✎ 편집 모드 · 눌러서 바로 수정      [⇅ 순서] [👁 미리보기]`
  테마 인식(gray-50 / dark #0E0E0E), 네이비 강제 배너 제거.
- **적립 strip → 한 줄 탭 가능 바**: `최근 30일 적립 ₩X (+예정) · 전환 X% · 콘솔→` (상세는 `/creator` 콘솔).
- 빈 링크샵(핀 0)은 기존 온보딩 빈 상태 그대로 (이미 깔끔).
- 방문자 뷰/미리보기 모드/주소·공유 카드/핀 카드 X·핀추가 dashed = **무변경**.
