# 🗺️ 동네딜 = 지도 + 바텀시트 (에버랜드 파크맵 스타일)

**시안 받은 날**: 2026-06-20 · **출처**: 대표 (에버랜드 앱 파크맵 스크린샷)
**상태**: ⏳ 구현 대기 (방향 확인 후) — **기존 자산 ~90% 재활용 가능**

## 시안 설명
에버랜드 공식 앱 파크맵 화면:
- 상단: 지도(시설 핀/캐릭터 마커) + 검색바 + 길찾기 버튼 + 현위치 버튼 + 새로고침
- 중앙: 위로 끌어올리는 **바텀시트 핸들(^)**
- 시트 상단: 가로 스크롤 **카테고리 칩** (공연 / 주토피아 / 레스토랑 / 편의시설 / 기프트샵 / 포토…)
- 시트 본문: 시설 **리스트 카드** (썸네일 + 이름 + 상태배지(CLOSED/금일종료) + 소요시간 + ➕ 담기)

대표 의도: "홈에 공구 상품들 있는 페이지를 이 첨부처럼" → **동네딜(오프라인 하이퍼로컬 공구)**을 지도 위 핀 + 바텀시트 리스트로.

## 🎉 핵심: 이미 거의 다 있음 — `RestaurantMapPage` (`/restaurant-map`)
`src/pages/RestaurantMapPage.tsx` + `src/pages/restaurant-map/*` 가 **정확히 이 패턴**을 이미 구현:
- Kakao 지도 (`useKakaoMap`)
- 드래그 바텀시트 + **snap 3단** (`sheetSnap: 'peek' | 'mid' | 'full'`)
- 카테고리/지역 필터 칩 (`SheetFilterBar`, `FilterSheet`)
- 리스트 (`RestaurantList`) + 선택 카드 (`SelectedPeekCard` / `SelectedDetailCard`)
- 검색 헤더 (`MapSearchHeader`) + **"내 주변" GPS** (`nearMeMode`, `userLoc`)
- **공구권 오버레이**: `useMapProducts` 로 식사/뷰티/숙소/헬스/펫/액티비티 공구권을 지도에 이미 올림

## 현재 vs 시안
| 항목 | 현재 | 시안/목표 |
|---|---|---|
| 패턴 존재 | ✅ `/restaurant-map` 에 완비 | 동일 |
| 진입 동선 | `/restaurant-map` (묻혀 있음) | **동네딜 메인 entry 로 승격** (`/group-buy` ↔ 지도뷰) |
| 데이터 | 공구권 + 카카오 맛집 | 동네딜 공구 상품에 **좌표(lat/lng)** 필요 — 매장 geocode 재활용 |
| 마커 | 맛집/공구권 | "동네딜 핀"(할인율/마감 배지) 정렬 |
| 테마 | 컬러 잔존 | **B&W(모노) 마커/칩**으로 정합 |
| 칩 | voucherType(식사/뷰티…) | 동네딜 카테고리 칩과 통일 |
