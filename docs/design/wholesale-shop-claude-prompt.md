# 유통스타트 도매몰 — Claude 디자인 생성용 프롬프트 (복붙용)

> 용도: Claude(아티팩트/디자인 생성)에 아래 "프롬프트"를 그대로 붙여넣어 고품질 도매몰 시안 생성.
> 기반 스펙: [`wholesale-b2b-mindmap.md`](./wholesale-b2b-mindmap.md), 명칭 확정: 플랫폼+도매몰.
> 확정 기본값: 비주얼=모던(쿠팡·스마트스토어 하이브리드), 장바구니 도입, 홈+상세+장바구니 전 화면 포함.

## 📋 그대로 복사해서 Claude 에 붙여넣으세요 ↓↓↓

---

당신은 시니어 프로덕트 디자이너 겸 프론트엔드 엔지니어입니다. **B2B 도매 쇼핑몰**의 고충실도 반응형 시안을 **React + Tailwind CSS 단일 아티팩트**로 만들어 주세요. 실제 동작하는 인터랙티브 목업(샘플 데이터 포함)으로, 모바일과 데스크톱 둘 다 보기 좋아야 합니다.

### 1. 서비스 개요
- 이름: **유통스타트 (UtongStart)** — "제조사와 유통사를 잇는 **B2B 유통 플랫폼**".
- 이 화면은 그중 **유통사(바이어)가 상품을 둘러보고 대량 주문하는 "도매몰"** 입니다.
- 핵심 가치: 유통사는 **자기 등급(C/B/A)에 맞는 공급가**로 검증된 제조사 상품을 사입합니다.

### 2. 사용자 & 목표
- 사용자: **로그인한 유통회원(도매 바이어, 소매상/유통사)**. 감성보다 **효율·신뢰·가격 판단**이 중요.
- 할 일(JTBD): "내 등급 공급가 확인 → 마진 계산 → 담기 → 한 번에 대량 주문".

### 3. 절대 규칙 (도메인 제약)
- **제조사(공급사) 이름·신원·원가는 절대 노출 금지.** 상품엔 "내 등급 공급가"만.
- **가격이 주인공**: 카드/상세에서 공급가를 가장 강하게. 권장소비자가는 취소선(마진 여력 암시).
- 모든 텍스트 **한국어**. 통화 "₩" + 천단위 콤마.
- **라이트 테마**. 브랜드 액센트색 **#FF0033**(레드). 보조 강조는 앰버(특가).
- 깔끔하고 정보 밀집된 모던 커머스 톤(쿠팡 / 네이버 스마트스토어 / 도매꾹 하이브리드).

### 4. 만들 화면 (3개, 탭/뷰 전환으로)

#### (A) 도매몰 홈
1. **상단 고정 헤더**:
   - 좌: 로고 "유통스타트" + 작은 배지 "도매몰"
   - 중앙: **검색바**("상품 검색")
   - 우: **내 등급 배지**(예: "C등급 · 마진 20%", 클릭 시 등급 안내) · 장바구니 아이콘(뱃지 숫자) · 주문내역 · 거래명세서 메뉴
2. **카테고리 가로 칩 네비**: 전체 / 식품 / 뷰티 / 리빙 / 패션 / 디지털 / 생활 (선택 시 강조)
3. **히어로 배너**(가로 캐러셀): 특가 프로모션 1~2장 (예: "🔥 이번주 임박 특가 — 최대 40%")
4. **🔥 특가·임박 섹션**(가로 스크롤 카드): 빨강/앰버 강조 + "D-2 23:59" 남은시간 칩. (기간한정 덤핑/임박 상품)
5. **📢 맞춤 추천**(가로 스크롤): "유통스타트가 회원님께 제안하는 상품"
6. **⭐ 베스트셀러**(그리드 일부) / **🆕 신상품**(가로 스크롤)
7. **메인 상품 그리드**(반응형: 모바일 2열 / 태블릿 3열 / PC 4~5열, 무한스크롤 느낌)
8. **모바일 하단 탭바**: 홈 / 카테고리 / 장바구니 / 주문내역 / 마이

#### (B) 상품 상세
- 좌: 이미지 갤러리(메인+썸네일). 우: 상품명 · 카테고리 칩 · **내 등급 공급가(아주 크게)** · 권장소비자가(취소선) · "마진 여력 ₩X" 보조표시 · 재고("재고 1,240개") · **최소주문수량(MOQ, 예: 10개)** · 수량 스텝퍼 · [장바구니 담기] [바로 주문] 버튼
- 하단 섹션 탭: 상세설명 / 배송 안내 / **정산 안내**("브랜드 상품: 익일 정산 / 일반 상품: 7일 정산") / 반품·교환
- "같은 카테고리 추천" 가로 스크롤

#### (C) 장바구니
- 담긴 상품 리스트(상품별: 썸네일·이름·등급공급가·수량 스텝퍼·소계·삭제)
- 우측/하단 요약: 상품 합계(공급가 기준) · 예상 배송 · **[주문하기(선결제)]**
- 보조: "엑셀로 대량 주문 업로드" 버튼(주문 많은 바이어용)

### 5. 상품 카드 스펙 (가장 중요)
```
┌───────────────────────┐
│ [정사각 이미지]            │  ← 좌상단 배지: "🔥특가" 또는 "-28%"
│                         │     우상단(선택): "달성"/"NEW"
├───────────────────────┤
│ 상품명 (최대 2줄)          │
│ ₩12,000   [C등급]        │  ← 공급가 굵게·크게(검정/빨강), 등급 작은 칩
│ 권장 ₩19,900             │  ← 회색 취소선
│ 재고 1,240 · 식품         │  ← 작은 메타
│ [−  10  +]   [담기🛒]     │  ← 수량 스텝퍼 + 담기 버튼
└───────────────────────┘
```
- 절대 제조사명 표시 X. 호버 시 살짝 떠오르는 느낌(shadow).

### 6. 샘플 데이터 (목업에 실제처럼 채워주세요)
유통사 등급: **C등급(마진 20%)**, 장바구니 3개 담김.
상품 예시(공급가=내 등급가 / 권장소비자가 / 재고 / 카테고리 / 배지):
1. 제주 감귤 2kg — ₩9,800 / ₩15,000 / 1,240 / 식품 / 🔥특가 D-1
2. 수제 약과 선물세트 — ₩12,500 / ₩22,000 / 430 / 식품 / -43%
3. 비건 핸드크림 50ml — ₩6,200 / ₩12,900 / 2,100 / 뷰티 / NEW
4. 스테인리스 텀블러 500ml — ₩7,400 / ₩14,900 / 880 / 리빙
5. 무선 충전 거치대 — ₩11,000 / ₩24,900 / 60 / 디지털 / 재고임박
6. 순면 오버핏 티셔츠 — ₩8,300 / ₩19,000 / 5,400 / 패션
7. 프리미엄 원두 1kg — ₩13,900 / ₩28,000 / 770 / 식품 / ⭐베스트
8. 실리콘 주방매트 — ₩4,900 / ₩9,900 / 3,200 / 리빙
(특가/베스트/신상품 섹션에 위 상품들 적절히 배치)

### 7. 반응형 & 인터랙션
- 모바일(≤640): 2열 그리드, 하단 탭바, 헤더 컴팩트. 데스크톱(≥1024): 좌측 카테고리 사이드바 가능 + 5열 그리드.
- 수량 스텝퍼·담기·장바구니 뱃지·탭 전환은 실제 동작(state)하게.
- 접근성: 버튼 aria-label, 충분한 대비.

### 8. 산출물
- **단일 React 컴포넌트(아티팩트), Tailwind CSS**. 외부 이미지가 필요하면 회색 플레이스홀더 박스 + 카테고리 이모지로 대체(실제 URL 불필요).
- 위 3개 화면을 **상단 세그먼트 토글(홈 / 상세 / 장바구니)** 로 전환해 한 아티팩트에서 다 보이게.
- 픽셀 디테일(여백·라운드·그림자·타이포 위계)까지 신경 써서 **바로 제품에 쓸 수 있는 수준**으로.

## 참고: 이 프롬프트가 반영한 시스템 사양 매핑
| 프롬프트 요소 | 스펙 근거 |
|---|---|
| 내 등급 배지(C/B/A·마진%) | 유통회원 가입=자동 C등급, 등급별 공급가 |
| 제조사 신원 비노출 | "제조회원은 공급가 외 확인 불가" + 관리자만 원가 |
| 🔥특가·임박 섹션 | 특별가격 등급 = 임박/덤핑, 선정 유통사만 |
| 📢 맞춤 추천 | 관리자→유통회원 상품 제안(proposals) |
| 정산 안내(브랜드 익일/일반 7일) | 정산기준 분기 |
| 엑셀 대량 주문 | 주문양식 다운/업로드 |
| 거래명세서/주문내역 헤더 | 자료 발행(거래명세서/세금계산서) |
| MOQ·재고 | 대량 거래 특성 |
