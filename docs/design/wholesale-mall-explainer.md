## 1. 한 줄 정의
**유통스타트(UtongStart)** = 검증된 **제조사**와 검증된 **유통사(소매 바이어)** 를 잇는 **B2B 도매 중개 플랫폼**.
양측은 서로 만나지 않고(신원·원가 비공개), 모든 거래·정산·세금계산서는 플랫폼을 경유.
- 도메인: **utongstart.com**(루트→`/wholesale`) / `live.ur-team.com/wholesale`(같은 코드·DB).
- 핵심 화면 = 유통사(바이어)가 도매 상품을 둘러보고 대량 사입하는 "도매몰".

## 2. 등장인물 3주체
| 주체 | 호칭 | 하는 일 | 진입 |
|---|---|---|---|
| 유통사(바이어) | 유통회원/판매사 | 등급 공급가로 무재고 사입 → 재판매 | `/wholesale` |
| 제조사(공급자) | 제조회원/공급회원 | 상품 등록 → 주문 발송(드랍쉽) → 정산 | `/supplier` |
| 어드민(운영자) | 유통스타트 | 가입 승인·최저가 검수·가격변경 승인·정산 지급·예치금/출금 관리 | `/admin/wholesale-*` |

## 3. 절대 도메인 규칙 (디자인이 틀리면 안 됨)
1. **제조사 신원·원가 노출 금지.** 유통사에게는 "내 등급 공급가"만.
2. **가격이 주인공.** 공급가 가장 강하게, 권장소비자가는 회색 취소선(마진 여력 암시).
3. **등급제 가격.** 가입 시 자동 **C등급(20%)**. 공급가 = `제조사 원가 × (1 + 등급마진율)`.
   - A 10% / B 15% / **C 20%(기본)** / D 25% / OEM 8% / 특별가 0%(기간 한정).
4. **결제 = 예치금.** 유통사가 예치금 충전(Toss) → 주문 시 차감. 대안: 여신/외상(`ON_CREDIT`).
5. **정산.** 제조사에게 브랜드 제품 **익일** / 일반 제품 **7일** 후 지급.
6. 모든 텍스트 **한국어**, 통화 **₩ + 천단위 콤마**.

```
ink  #17181C 근블랙   ink2 #4E5560 보조   ink3 #8A929E 캡션   ink4 #B6BCC4 흐림
line #ECEEF1 보더     fill #F4F5F7 섹션배경  fill2 #F8F9FB 카드내부
brand #FF0033 주 CTA·활성·핵심단가  brandSoft #FFF0F2   pos #11875A 마진/이득(초록)
```
- 액센트는 **#FF0033 단 하나**. 파랑/초록 임의 사용 금지(마진 표시 `pos` 초록만 예외).
- 카테고리: 전체/식품/뷰티/리빙/패션/디지털/생활.

## 6. `/wholesale` 홈 구조 (2026-06-15 로그인 분기 정리 반영)
- **비로그인 방문자 = 마케팅(전환):** 배너 캐러셀 → 가입 유도 → 발견 레일(베스트셀러·신규입고) → 상품 그리드 → BrandHero(서비스 정체성).
- **로그인 사입자 = 빠른 카탈로그:** 슬림 사입 대시보드(등급·예치금·이번달 사입액) → 개인화 레일(빠른 재주문·전용 공급) → 바로 상품 그리드. (마케팅·중복 베스트 레일·대량주문 패널 숨김/접힘).
- **헤더:** ① 유틸바(데스크톱은 카테고리 네비 우측으로 흡수 — 2단) ② 로고+중앙 검색바+우측 아이콘(처음이세요/제안·신고/예치금) ③ 카테고리 네비(전체카테고리 메가메뉴 + 브랜드전시관·월간베스트·신상품·판매마진·프리미엄·공지).
