## 🔴 전체 적용 — 용어 통일 (사이트 전역)
사이트 곳곳 단어가 안 맞음 → **통일**:
- **판매사(유통사)** = 유통회원/유통사/판매사 → "판매사(유통사)"
- **제조사(공급사)** = 제조회원/공급회원/브랜드사 → "제조사(공급사)"
- 로그인/가입 링크: "유통회원 로그인" → "판매사(유통사) 로그인", "제조회원 로그인" → "제조사(공급사) 로그인"

- [ ] 카테고리 네비 **맨 앞에 "전체상품" 추가** (브랜드 전시관 앞)

## 2. 게스트 홈 히어로 (HeroSection)

- [ ] 히어로 배너는 **제품 관련 / 플랫폼 공지** 용도로 활용

- [ ] 우측 "오늘의 도매 특가" 카드 **삭제**
