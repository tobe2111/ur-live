## 소싱스타트 제조회원 유통 지원 B2B 도매몰

### 사전 (운영 프로세스)
- 제품 카테고리 결정
- 제조회사 컨택 & 회원가입 & 상품등록 요청
- 제품 카테고리 유통회사 선별 → 회원가입 요청

## 1. 제조회사 (제조회원)

### 제조회원 가입
### 제품 등록
- **개별 등록**
- **대량등록** — 엑셀(가능여부 체크) + 표준 양식 준비
- **수정** — 공급가격 수정 시 **수정 전 금액 기록(관리자만 확인)**
- **공급가격 등급별 가격** — A/B/C, 특별가격 등급 총 **4단계**
  - 특별가격 등급은 가격 입력 후 **관리자에서 선정한 유통회사만** 가격 확인·유통 가능
- **공급가격 노출 기준**
  - 유통회원에게 **C 등급** 공급가격 노출
  - 유통회원에 공급하는 공급가격은 **관리자만 관리/확인**
  - 제조회원은 유통스타트에 공급하는 공급가격 외 확인 불가
- **OEM/ODM 지원 가능 항목 선택**
- **'바코드' 입력 항목** — 오프라인 유통채널 판로 확장용

### 제품 공급 여부 승인 (제조회원 가입 시 선택)
- **전체 공급**
- **승인한 유통채널 공급**
- **유통스타트 유통채널 공급**
  - 전체 공급이지만 신규 상품(선정 제품)을 유통스타트 유통채널 공급으로 상품등록 시 선택 → **선정된 유통회원에게만 상품 공개**

### 주문 & 송장번호 입력
- **주문 확인**
  - (많을 경우) 주문양식 다운(엑셀) — 표준 주문양식 준비
  - (적을 경우) 주문확인 (유통스타트 플랫폼에서)
- **송장번호 입력**
  - (많을 경우) 주문양식 다운(엑셀) → 송장번호 입력 후 업로드
  - (적을 경우) 주문별 입력

### 반품 & 교환

### 정산
- **정산기준** (제조회원 등록 시 선택)
  - **(브랜드 제품)** 정산 후 배송 / 판매 후 당일 정산
  - **(일반 제품)** 월마감 정산 또는 15일(7일) 정산으로 유도 (정산일 다양하면 업무 과다)
  - 유통스타트 또는 유통회사 기준에 맞춰 정산
- (일반 제품) 정산 기준 **세금계산서 발행(청구) 확인 후 정산**

---

## 2. 유통회사 (유통회원)

### 유통회원 가입 — **기본 C등급**
### 제품확인
- (기본) 제조회원 가입 시 선택(전체 공급/승인한 유통채널 공급)한 제품 확인
- 제조회원이 '유통스타트 유통채널 공급' 선택한 제품: (기본) + **관리자로부터 선정되면** 제품 확인
### 주문
- (많을 경우) 주문양식 다운(엑셀) → 주문서 작성 후 업로드
- (적을 경우) 주문신청 (유통스타트 플랫폼에서)
### 송장번호
- (많을 경우) 주문양식 다운(엑셀) — 송장번호 포함
- (적을 경우) 송장번호 확인 (유통스타트 플랫폼에서)
### 반품 & 교환
### 정산
- **(브랜드 제품)** 브랜드 제품 정산 기준에 맞춰 가능할 경우 주문 가능
- **(일반 제품)** 유통스타트에서 정한 정산 기준에 맞추면 됨 / 정산 기준보다 빠르게 정산 시 OK

## 3. 관리자

### 제조회원
- 등급이 없음
- OEM 지원 가능 제조회원 구분 필요(별도 관리)
- 제조회원 가입 시 선택: 유통스타트 유통채널 공급으로 선택한 제조회원 → 유통회원 관리(선정/삭제)

### 상품관리
- **특별가격 상품 별도 관리** — 기간 확인/연장(날짜 선택)/중지 + 특별가격 선택 후 가격 입력 → 해당 특별가격 유통할 유통회원 관리(선정/삭제)
- 전체 공급으로 선택했어도 신규/특별 상품을 '유통스타트 유통채널 공급' 제품으로 등록 → 유통할 유통회원 관리(선정/삭제)

### 등급
- 유통회사 A/B/C 등급에 맞춰 공급가격 %에 맞춰 **자동 입력**

### 공급가격
- 등록 상품 중 특별가격 선택(체크) 시 특별가격 항목에 가격 수동 입력 등록 — 특별한 제품 / 임박·덤핑 제품 위주
- A/B/C + 특별가격 등급 총 4개 등급 (공급가격 등급)
- 특별가격 등급 회원: 제조회원이 선택(체크)·가격 입력 → 관리자가 유통회원 선정

### 유통회원
- OEM 신청은 별도 관리 — (제품 생산 후) 제조회원으로 가입 유도
- **유통회원 가입하면 자동 C등급. 관리자가 등급 조절** → 등급에 해당하는 공급가격으로 유통(판매) 후 정산 가능
- (검토 후 반영) 유통회원이 판매가격(최저가격) 대비 받고 싶은 공급가격 %를 적어 제조회사 공급가격 매칭 지원 후 등급 반영 검토

### 상품정보
- 엑셀로 다운받아 수정·유통채널에 제안 가능하면 좋음
- 상품공급가격기준: 기본 C등급 / A·B등급 공급가격으로도 다운 가능

### 정산
- 제조회원: 입금하기 → 확인
- 유통회원: 입금받기 → 확인
- **사람과고리는 1일 1억까지 정산 가능**

### 자료 발행
- **세금계산서**: To 유통회원 / From 제조회원
- **거래명세서**: To 유통회원

### 제품 공급 여부 승인
- 유통스타트 유통채널 공급: 제조회원 가입 시 선택 가능 + 제품 등록 시 선택 가능
- 관리자 유통회원 선정

### OEM/ODM 지원 가능 항목 선택
- 유통회원 중 OEM/ODM 통해 자사 브랜드 제품 필요한 회사 대상 — 제조회사 연결과 컨설팅 지원
- OEM/ODM 신청 → 유통스타트가 제조사 찾기 → 유통회사 연결 → 제품 생산까지 지원
- (추후) OEM/ODM 지원 가능한 상품 목록 페이지만 별도 지원
