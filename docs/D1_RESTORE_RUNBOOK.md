## 평시 (자동 — 할 일 없음)
- D1 이 모든 변경을 자동 보존 (30일). 추가 백업 작업 불필요.

## 🔥 사고 시 복구 절차 (예: 데이터 오염/실수 삭제)
1. **즉시 쓰기 중단**: Cloudflare Dashboard → Workers & Pages → ur-live → 비활성화(또는 main 배포 중단). 오염이 계속 쌓이는 것 방지.
2. **복구 시점 결정**: 사고 직전 시각 (KST → UTC 변환 주의).
3. **북마크 확인** (현재 상태 보존 — 복구 잘못 시 되돌리기용):
   ```bash
   npx wrangler@4 d1 time-travel info <DB이름>
   ```
4. **복구 실행**:
   ```bash
   npx wrangler@4 d1 time-travel restore <DB이름> --timestamp="2026-06-10T14:00:00Z"
   ```
   (또는 `--bookmark=<북마크>` 로 정확 복원)
5. **검증**: `/admin/health` + 핵심 페이지(상품/주문) 확인 → 서비스 재개.
6. **주의**: 복구 시점 이후의 정상 주문/결제도 사라짐 — Toss 대시보드와 대조해 누락 주문 수동 보정 (orders 테이블 + 알림).

## DB 이름 확인
