## 📋 역할 분담 (최종)

### **1. Firebase Authentication (🆕 추가 권장)**
**역할:** 사용자 인증 및 보안
- ✅ 이메일/비밀번호 로그인
- ✅ 소셜 로그인 (카카오, Google, Facebook 등)
- ✅ JWT 토큰 자동 발급/검증
- ✅ 이메일 인증, 비밀번호 재설정
- ✅ 2FA/MFA 지원

**선택 이유:**
- 보안 자동화 (bcrypt, JWT, CSRF 방어)
- 개발 시간 90% 절감
- 무료 (월 10,000명)
- 소셜 로그인 간편 추가

**비용:** 무료 (Spark Plan)

---

### **2. Firebase Realtime Database (현재 사용 중)**
**역할:** 초저지연 실시간 데이터
- ✅ 채팅 메시지 (0.2초)
- ✅ 시스템 알림 (0.2초)
- ✅ 재고 업데이트 (0.2초)
- ✅ 시청자 수 (0.2초)
- ✅ 상품 변경 (0.2초)

**선택 이유:**
- 초저지연 필수 (0.2초)
- 양방향 실시간
- 서버 부하 없음
- 자동 확장

**비용:** 무료 (Spark Plan)

---

### **3. Cloudflare D1 Database (현재 사용 중)**
**역할:** 영구 저장소 및 복잡한 쿼리
- ✅ 사용자 정보 (firebase_uid 연결)
- ✅ 주문/결제 데이터 (ACID 트랜잭션)
- ✅ 상품 마스터 데이터
- ✅ 장바구니
- ✅ 라이브 스트림 정보

**선택 이유:**
- ACID 트랜잭션 보장
- 복잡한 쿼리 (JOIN, 필터링)
- 검색/분석
- 영구 저장

**비용:** 무료 (첫 5GB)

---

### **4. Cloudflare Workers (현재 사용 중)**
**역할:** API 서버 및 비즈니스 로직
- ✅ RESTful API 제공
- ✅ Firebase JWT 검증 (🆕)
- ✅ 결제 처리 (Toss Payments)
- ✅ 주문 관리
- ✅ D1 ↔ Firebase 동기화

**선택 이유:**
- Edge 배포 (초고속)
- 서버리스 (자동 확장)
- 비용 효율

**비용:** 무료 (첫 100k 요청/일)
