# 라이브 송출 인프라 로드맵

**최종 업데이트**: 2026-05-16
**위치**: 차세대 세션이 "왜 자체 RTMP 안 했지?" 다시 묻지 않도록 박제.

## 현 단계 — Phase 1 (시드/MVP)

**활성 셀러 < 100명, 매출 미정착.**

### 송출 경로
```
[셀러 PC/모바일]
  PRISM Live Studio / OBS / 우리 퀵 라이브 (브라우저) / 우리 모바일 앱
    ↓ 셀러가 선택
  YouTube Live (셀러 본인 채널)
    ↓ youtube_video_id
  유어딜이 iframe embed (`<iframe src="youtube.com/embed/...">`)
    + 자체 SSE 채팅 + 상품 카드 오버레이 + 결제
```

### 왜 YouTube embed
- **트래픽 비용 ₩0** — Google CDN 사용
- 화질 안정성 ⭐⭐⭐⭐⭐ (Google ABR)
- 동시접속 사실상 무제한
- 다시보기 자동 보관 (~24h+)
- 폰 인증만 하면 셀러 즉시 라이브 가능

### 한계 (수용)
- 영상 지연 10~30초 (Ultra-low latency 설정 시 2~5초)
- iframe 자유도 제한 (오버레이만 가능)
- YouTube 광고 노출 가능 (monetization OFF 권장)

## Phase 2 (그로스) — 활성 셀러 100~500명, 다이아 등급 5명+

### 트리거
- 다이아 등급 셀러 (월 매출 1,000만원+) 5명 이상
- 인터랙티브 라이브 (경매/PK/1:1 후원) 매출 비중 > 20%

### 변화: 하이브리드
```
[일반/실버/골드/브론즈/뉴 셀러]
  → 그대로 YouTube embed

[다이아 셀러 — 옵션 제공]
  → OvenMediaEngine 자체 RTMP 송출 (선택)
  → WebRTC 1초 지연 + 자체 player + 광고 0
  → 다이아 수수료 % 인상으로 인프라 비용 상쇄
```

### 예상 인프라 비용
- OME self-host (EC2 c5.large + Bunny CDN): **월 $1,500~$3,000**
- 다이아 셀러 5명 × 월 매출 1,000만원 × 0.5% 추가 수수료 = 월 250만원 → ROI 충분

## Phase 3 (스케일) — 활성 셀러 500명+

### 트리거
- mallpro 수준 셀러풀
- 자체 광고/스폰서 수익 모델 추가
- B2B 영업 무기 필요 ("자체 인프라" 어필)

### 변화: 전면 자체 송출
```
모든 셀러 PRISM/OBS → 우리 자체 RTMP ingest (Cloudflare Stream Live 또는 self-host)
  → 자체 player + 자체 채팅 + 자체 결제 통합
  → YouTube 임베드 옵션은 신규/체험 셀러만
```

### 예상 인프라 비용
- Cloudflare Stream Live: **월 $5,000~$15,000** (활성 셀러 500명 × 일 5시간)
- 또는 self-host OME + 자체 CDN: **월 $3,000~$8,000 + 인력**

## mallpro 비교 분석 (2026-05-16)

mallpro 가 자체 RTMP 운영하는 12가지 이유 (요약):

1. 광고 노출 0 약속 가능 (B2B 영업)
2. 시청자 락인 (YouTube 앱으로 이탈 0)
3. 시청 데이터 100% 소유 (ML/타깃팅)
4. 셀러 진입장벽 제거 (YouTube 폰 인증 불필요)
5. YouTube 정책 변경 리스크 회피
6. 브랜딩 (YouTube 로고 안 뜸)
7. UI/UX 풀 제어
8. 자체 광고/스폰서 수익
9. 저작권 음악 자유도
10. VOD/다시보기 정책 자유
11. 글로벌 라이센싱 회피
12. **이미 매출/자본이 받쳐줌** ← 가장 중요

→ **우리는 Phase 1, mallpro 는 Phase 3.** 같은 무대 다른 시간.

## PRISM 사용 가이드 (Phase 1 일관)

PRISM 자체는 우리 인프라/비용에 무관. 셀러가 편한 도구 선택:

| 셀러 유형 | 권장 도구 |
|---|---|
| 자취방/카페 1인 셀러 | 우리 퀵 라이브 (브라우저) 또는 모바일 앱 |
| 본인 채널도 키우려는 셀러 | PRISM (multi-stream — YouTube 외 본인 IG/TikTok 추가 가능) |
| 게이밍/방송러 (이미 OBS 사용) | OBS |
| 다이아 등급 (Phase 2 이후) | OBS + 우리 자체 RTMP (1초 지연) |

### 셀러 가이드 권장 (반드시 안내)

- **YouTube Studio "Ultra-low latency" 설정** → 지연 30초 → 3초
- **monetization OFF** → 광고 노출 방지
- **네이버 쇼핑라이브 동시 송출 금지** (직접 경쟁사 매출 누수) — 약관에 명시 필요
- TikTok Live / Instagram Live = 본인 채널 성장용. 우리 사이트 임베드는 YouTube 만.
