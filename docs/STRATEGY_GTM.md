# 마케팅·세일즈 전략 (GTM) — 2026-06-10 초안

> 전제: 제품(퍼널 장치)은 거의 완성 — 예치금/등급공급가/게시판/자료실/최저가신고/찜/약관(도매),
> ➕제안 루프/현장 스캔/자동 정산/교환권/링크샵 적립(동네딜). **이제 병목은 코드가 아니라 영업.**
> 참고: docs/reference/competitor-sellpie-analysis.md (셀파이 — 2년만에 31억, 셀러 2천)

## 원칙
1. **유료 광고 금지(당분간)** — 퍼널 전환이 검증되기 전 광고비는 소각. 무료 채널 + 직접 영업만.
2. **시간 배분**: 개발 20% / 영업·콘텐츠 80% (제품 추가는 영업에서 막힌 것만).
3. **두 사업 모두 "공급 먼저"**: 팔 것(제조사 상품/매장 공구)이 없으면 수요 모집은 무의미.

## A. 도매몰 (유통스타트)

### 포지셔닝
- "전 카테고리 도매몰" ❌ → **"국내 ○○ 카테고리 1위 B2B 위탁 도매"** ⭕ (셀파이의 핵심 비결).
- 첫 몰 카테고리 1개 선정(예: 식품/생활) → 검증 후 멀티몰 복제(인프라 이미 있음).
- 카피에 숫자 고정: **"판매마진 최대 40% · 무재고 위탁 · 예치금 간편결제 · 세금계산서 자동"**.

### 세일즈 순서 (공급 → 수요)
1. **제조사 확보 (선행, 직접 영업)** — 목표: 제조사 10곳 / 상품 100개
   - 타깃: 오너클랜·도매매 입점 중소제조사, 카테고리 협회/박람회 명단, 스마트스토어 직영 제조사.
   - 제안서 1장: "무료 입점 · 수수료 외 비용 0 · 직발송만 하면 정산·세금계산서·셀러 모집 전부 플랫폼이".
   - 도구: /supplier 가입 데모 + 채팅 + 자동 정산 화면 시연.
2. **유통셀러 모집 (기수제)** — 목표: 1기 50명
   - **"유통스타트 1기 프리미엄 셀러 ○명 한정 모집"** (셀파이 검증 퍼널 — 5기까지 운영 중). 혜택: 상위 등급 공급가 6개월 + 자료실 전 자료 개방.
   - 채널(무료): 스마트스토어 카페(아프니까 사장이다 등), dcinside 스마트스토어 갤러리(셀파이도 여기서 모집), 위탁판매 오픈카톡, 블로그 "위탁판매 시작" SEO.
   - 가입 즉시 가치: 자료실 이미지 다운로드 → 당일 스토어 등록 가능.
3. **리텐션**: 공지사항=시즌 상품 푸시, 최저가 신고=가격 신뢰, 재입고 알림/찜=재방문.

### KPI (월)
제조사 수 → 등록 상품 수 → 승인 셀러 수 → **첫 주문 전환율** → 월 거래액 → 셀러당 재주문율.

## B. 동네딜 (오프라인 공구 + 교환권)

### 포지셔닝
- 소비자: "우리 동네 맛집, 같이 사면 더 싸다" (홈 태그라인 적용됨).
- 사장님: **"선결제 현금 먼저 + 새 단골" — 광고비가 아니라 매출 선입금**.

### 세일즈 순서
1. **지역 1곳 집중** (동/구 단위) — 밀도가 생명. 전국 분산 ❌.
2. **사장님 직접 영업** — 목표: 매장 5곳/월
   - 도구: 셀러 스캔(/seller/scan) 현장 데모 + "발행 5분, 정산 자동, 수수료 5%" 원장 시연.
   - 우선 타깃: 카페/식당 중 인스타 운영하는 곳(디지털 수용도↑).
3. **수요 확산 (무료)**
   - ➕ 동네 공구 제안 루프(구축됨) → 제안 모이면 그 가게부터 영업(수요 검증된 영업 리스트).
   - 당근마켓 동네생활/지역 맘카페에 공구 오픈 글, 매장 테이블 QR.
   - **링크샵 = 분산 유통**: 주민/인플루언서가 자기 링크샵에 교환권 핀 → 적립(레일 구축됨). "공유하면 적립" 한 줄이 마케팅 조직을 대체.
4. **교환권 = 현금흐름**: 선결제라 정산 전 현금 보유 — 공구 1건당 현금 회전 강조.

### KPI (월)
제휴 매장 수 → 오픈 공구 수 → 공구당 참여자 → ➕제안 수/전환율(30일 판정 지표) → 교환권 사용률 → 재구매율.
