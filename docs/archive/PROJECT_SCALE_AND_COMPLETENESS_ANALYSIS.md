## 🎯 프로젝트 개요

### 서비스 설명
**유어라이브**는 실시간 라이브 방송과 상품 판매를 결합한 **풀스택 라이브 커머스 플랫폼**입니다.

- **비즈니스 모델**: B2C + B2B2C (플랫폼 중개형)
- **핵심 가치**: 실시간 라이브 스트리밍 + 즉시 구매 + 안전한 결제
- **타겟 시장**: 한국 라이브 커머스 시장 (2026년 기준 약 10조원 규모)

### 주요 사용자 유형 (3가지)
1. **일반 구매자** (C) - 라이브 시청 + 상품 구매
2. **셀러/판매자** (B) - 라이브 진행 + 상품 관리 + 정산 조회
3. **관리자** (Admin) - 플랫폼 운영 + 셀러 승인 + 정산 관리
