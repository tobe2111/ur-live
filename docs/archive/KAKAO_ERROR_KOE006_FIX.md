# KOE006 에러 해결 가이드

## 에러 메시지
```
앱 관리자 설정 오류 (KOE006)
유어팀 라이브 서비스 설정에 오류가 있어, 이용할 수 없습니다.
서비스 관리자의 확인이 필요합니다.
```

## 원인
- 카카오 로그인이 활성화되지 않음
- Web 플랫폼이 등록되지 않음
- Redirect URI가 정확하지 않음
- 동의항목이 설정되지 않음

---

## 해결 방법 (순서대로 진행)

### 1️⃣ 카카오 로그인 활성화

**위치:** Kakao Developers > 내 애플리케이션 > 카카오 로그인

**설정:**
```
1. 왼쪽 메뉴에서 "카카오 로그인" 클릭
2. 상단의 "활성화 설정" 스위치를 ON으로 변경
3. 자동 저장됨
```

**확인:**
- 활성화 설정 스위치가 **초록색 ON** 상태여야 함

---

### 2️⃣ Web 플랫폼 등록

**위치:** Kakao Developers > 내 애플리케이션 > 플랫폼

**설정:**
```
1. 왼쪽 메뉴에서 "플랫폼" 클릭
2. "Web 플랫폼 등록" 버튼 클릭
3. 사이트 도메인 입력:
   - https://live.ur-team.com
4. "저장" 버튼 클릭
```

**추가 등록 (선택):**
```
- http://localhost:3000 (로컬 개발용)
```

**확인:**
- 플랫폼 목록에 "Web"이 표시되어야 함
- 사이트 도메인: https://live.ur-team.com

---

### 3️⃣ Redirect URI 정확히 등록

**위치:** Kakao Developers > 내 애플리케이션 > 카카오 로그인

**설정:**
```
1. 왼쪽 메뉴에서 "카카오 로그인" 클릭
2. 아래로 스크롤하여 "Redirect URI" 섹션 찾기
3. "등록" 버튼 클릭
4. URI 정확히 입력:
   https://live.ur-team.com/auth/kakao/callback
5. "저장" 버튼 클릭
```

**추가 URI (로컬 개발용):**
```
http://localhost:3000/auth/kakao/callback
```

**⚠️ 주의사항:**
- 대소문자 정확히 일치
- 슬래시(/) 위치 정확히 일치
- http:// 또는 https:// 포함
- 끝에 슬래시(/) 없음

**확인:**
- Redirect URI 목록에 2개 URI가 모두 표시되어야 함

---

### 4️⃣ 동의항목 설정

**위치:** Kakao Developers > 내 애플리케이션 > 카카오 로그인 > 동의항목

**설정:**
```
1. 왼쪽 메뉴에서 "카카오 로그인" 클릭
2. "동의항목" 탭 클릭
3. "닉네임" 항목 찾기
4. "필수 동의" 또는 "선택 동의" 활성화
5. "설정" 버튼 클릭
```

**권장 설정:**
```
- 닉네임: 필수 동의
- 카카오계정(이메일): 선택 동의 (수집 목적: 회원 식별)
- 프로필 사진: 선택 동의
```

**확인:**
- 닉네임 항목이 "활성화" 상태여야 함

## ✅ 최종 확인 체크리스트

모든 설정을 완료한 후 다음을 확인하세요:

```
□ 카카오 로그인 > 활성화 설정: ON (초록색)
□ 플랫폼 > Web 플랫폼: https://live.ur-team.com 등록됨
□ 카카오 로그인 > Redirect URI:
  □ https://live.ur-team.com/auth/kakao/callback
  □ http://localhost:3000/auth/kakao/callback
□ 카카오 로그인 > 동의항목 > 닉네임: 활성화
□ 앱 키 > REST API 키: 5dd74bccb797640b0efd070467f3bafd
```

## 🔍 여전히 에러가 발생한다면?

### 확인 사항:

**1. 앱 상태 확인**
- 내 애플리케이션 > 앱 설정 > 일반
- 앱 상태가 **"개발 중"** 또는 **"서비스 중"**인지 확인

**2. 비즈 앱 전환 (선택)**
- 일부 기능은 비즈 앱에서만 작동
- 앱 설정 > 비즈 앱 전환 고려

**3. Kakao 고객센터 문의**
- https://devtalk.kakao.com
- 앱 키와 함께 문의
