## 🎯 서비스 개요

### UR Live는 무엇인가요?

**UR Live**는 YouTube와 TikTok 같은 소셜 미디어 영상을 활용한 **차세대 라이브 커머스 플랫폼**입니다. 판매자는 기존에 제작한 영상 콘텐츠를 그대로 활용하여 별도의 방송 장비나 기술 없이도 즉시 라이브 커머스를 시작할 수 있습니다.

### 해결하는 문제

**판매자의 고민:**
- ❌ 라이브 방송을 위한 장비와 기술 부족
- ❌ 실시간 방송 진행의 부담과 시간 제약
- ❌ 높은 마케팅 비용과 진입 장벽
- ❌ 기존 영상 콘텐츠 활용의 어려움

**구매자의 불편:**
- ❌ 긴 상품 설명을 읽어야 하는 번거로움
- ❌ 정적인 이미지만으로는 제품 파악 어려움
- ❌ 실시간 질문과 소통의 제약
- ❌ 복잡한 결제 프로세스

### UR Live의 솔루션

**판매자에게:**
- ✅ YouTube/TikTok 영상 URL만 입력하면 즉시 라이브 커머스 시작
- ✅ 별도의 방송 장비나 기술 없이 간편한 운영
- ✅ 24시간 자동 운영되는 무인 라이브 커머스
- ✅ 실시간 채팅으로 고객과 소통 (선택적)

**구매자에게:**
- ✅ 영상을 보면서 바로 구매 결정 가능
- ✅ 실시간 채팅으로 즉각적인 질문과 답변
- ✅ 간편한 1-2단계 결제 프로세스
- ✅ 안전한 결제 시스템 (TossPayments 연동)

## 💎 핵심 가치

### 1. **간편함 (Simplicity)**
> "YouTube/TikTok URL만 있으면 5분 안에 라이브 커머스 시작"

- 복잡한 설정 없이 영상 URL만 입력
- 직관적인 대시보드로 누구나 쉽게 관리
- 자동화된 주문/배송 관리 시스템

### 2. **접근성 (Accessibility)**
> "누구나, 어디서나, 언제든지"

- 모바일 최적화로 언제 어디서나 접속 가능
- 카카오 로그인으로 간편 가입 (1초 만에 시작)
- 판매자 승인 절차 간소화 (24시간 내 승인)

### 3. **경제성 (Affordability)**
> "높은 수수료 부담 없이 시작하는 라이브 커머스"

- 초기 비용 ZERO (무료 시작)
- 판매 수수료만 부담 (업계 최저 수준)
- 별도의 방송 장비나 스튜디오 불필요

### 4. **신뢰성 (Trustworthiness)**
> "안전한 거래와 투명한 정산"

- TossPayments 공식 연동 (PCI-DSS 인증)
- 자동 세금계산서 발행 시스템
- 투명한 정산 내역 실시간 확인

## 🚀 주요 기능

### 🎬 **라이브 스트리밍 커머스**

#### YouTube & TikTok 영상 연동
```
1. 판매자 대시보드 접속
2. 라이브 방송 생성 클릭
3. YouTube/TikTok 영상 URL 입력
4. 상품 연결 및 가격 설정
5. 즉시 라이브 방송 시작! ✅
```

**특징:**
- 🎥 YouTube 영상: `youtube_video_id` 자동 추출
- 🎵 TikTok 영상: 임베드 플레이어 지원
- 📱 모바일 최적화: 세로 영상 (9:16) 완벽 지원
- 🔄 자동 재생: 영상 종료 시 자동 반복
- 👥 실시간 시청자 수: 현재 접속자 카운트 표시

#### 실시간 채팅 시스템
- 💬 **실시간 메시지**: 3초마다 자동 새로고침으로 실시간 소통
- 🚫 **욕설 필터링**: AI 기반 비속어 자동 차단
- 📏 **메시지 제한**: 500자 이내 (스팸 방지)
- 🏷️ **뱃지 시스템**: 
  - 판매자 (Seller) 뱃지
  - 관리자 (Admin) 뱃지
- 🗑️ **메시지 관리**: 관리자 권한으로 부적절한 메시지 삭제

#### 시청자 수 관리
- 👥 **실시간 카운트**: 현재 라이브 시청 중인 사용자 수
- 📊 **자동 증가**: 사용자 입장 시 자동 +1
- 🎛️ **수동 조정**: 어드민/셀러 대시보드에서 조정 가능
- 📈 **통계 대시보드**: 시간대별 시청자 수 추이 분석

### 🛒 **쇼핑 기능**

#### 상품 관리
**판매자 기능:**
- ➕ 상품 등록: 이름, 가격, 설명, 이미지, 재고 관리
- 📸 **상품 이미지 갤러리**: 최대 10개 이미지 업로드 지원
- 🏷️ 할인 설정: 할인율 자동 계산 및 표시
- 📦 재고 관리: 실시간 재고 추적 및 알림
- 📊 판매 통계: 상품별 판매량, 매출 분석

**구매자 경험:**
- 🔍 **상품 검색**: 
  - 자동완성 (2글자 이상 입력 시)
  - 정렬: 관련도순, 가격순, 최신순
  - 필터: 가격 범위 설정
- 📱 **상품 상세 페이지**:
  - 이미지 갤러리 (스와이프 지원)
  - 상세 설명 및 스펙
  - 판매자 정보 및 리뷰
- 🛒 **장바구니**:
  - 다중 상품 담기
  - 수량 조절
  - 쿠폰 적용
- ❤️ **찜하기**: 관심 상품 저장 및 알림

#### 결제 시스템
**TossPayments 공식 연동:**
- 💳 **결제 수단**:
  - 신용/체크카드
  - 계좌이체
  - 간편결제 (카카오페이, 네이버페이, 토스페이 등)
  - 휴대폰 소액결제
- 🔒 **보안**:
  - PCI-DSS Level 1 인증
  - 3D Secure 인증
  - SSL/TLS 암호화
- 📱 **모바일 최적화**:
  - Intent URL 자동 처리
  - 앱 자동 전환 및 폴백
  - localStorage 백업으로 안정적인 결제

**결제 프로세스:**
```
1. 상품 선택 → 장바구니 담기
2. 결제하기 클릭
3. 배송지 입력/선택
4. 결제 수단 선택
5. 결제 승인
6. 주문 완료! 🎉
```

```
│ 마케팅 (SNS 광고)              │ 변동          │
└────────────────────────────────┴──────────────┘
```

### 손익분기점 (Break-Even Point)

**월 손익분기점 계산:**
```
고정 비용: ₩56,000 + ₩1,000,000 = ₩1,056,000

필요 플랫폼 수수료: ₩1,056,000
거래액: ₩1,056,000 ÷ 5% = ₩21,120,000

결론: 월 거래액 약 2,100만 원 달성 시 손익분기점 도달
```
