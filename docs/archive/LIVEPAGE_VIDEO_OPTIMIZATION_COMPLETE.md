### ✅ 2. 상품 카드를 결제 버튼 옆으로 이동 (이미지 형태)
**요구사항**: 
- 제공된 이미지처럼 상품 카드를 결제 버튼 옆에 배치
- 새로 만든 "상품" 버튼 제거
- 상품 카드를 최대한 아래로 이동하여 영상 노출 공간 확대

**구현**:
```typescript
// 하단 버튼 - 상품 카드 + 결제 버튼
<div className="flex gap-3 items-center">
  {/* 상품 카드 (이미지 형태) */}
  <button className="flex-1 flex items-center gap-3 px-3 py-2 rounded-2xl">
    <img src={image_url} className="w-14 h-14" />
    <div className="flex-1">
      <p className="text-[11px]">{product.name}</p>
      <div>
        <span className="text-[11px]">{discount_rate}%</span>
        <span className="text-[13px]">{price}원</span>
      </div>
    </div>
    <div className="bg-[#FF6B35]">담기</div>
  </button>

  {/* 결제 버튼 */}
  <button className="flex-shrink-0">
    <ShoppingBag />
    <span>결제</span>
  </button>
</div>
```

**디자인**:
- **상품 카드**: 
  - 흰색 배경 (white/95)
  - 왼쪽 상품 이미지 (56px × 56px)
  - 가운데 상품명 + 가격
  - 오른쪽 주황색 "담기" 버튼
  - 전체 버튼으로 클릭 가능
- **결제 버튼**:
  - 파란색 배경 (#0064FF)
  - 고정 너비 (flex-shrink-0)
  - 장바구니 아이콘 + "결제" 텍스트

**레이아웃**:
```
┌────────────────────────────────────────┐
│  [상품 이미지] 상품명     [담기]       │ 70%
│              가격                      │
└────────────────────────────────────────┘
                                  ┌──────┐
                                  │  결제 │ 30%
                                  └──────┘
```

### ✅ 3. 채팅 메시지 간격 축소
**문제**: 채팅 메시지 간 간격이 너무 넓어서 화면 공간 낭비

**변경 사항**:
```typescript
// Before
<div className="space-y-2 max-h-28">  // 간격 8px
  <div className="px-3 py-1.5">       // 패딩 12px, 6px
    ...
  </div>
</div>

// After
<div className="space-y-1 max-h-24">  // 간격 4px (-50%)
  <div className="px-2.5 py-1">       // 패딩 10px, 4px (-33%)
    ...
  </div>
</div>
```

**개선 결과**:
- 메시지 간 간격: 8px → 4px (**-50%**)
- 메시지 패딩: 12px/6px → 10px/4px (**-33%**)
- 채팅 영역 높이: 112px → 96px (**-14%**)

## 🎯 핵심 개선 사항

### 1. 영상 노출 극대화
- 상품 카드를 하단으로 통합
- 채팅 간격 축소
- 2줄 레이아웃 → 1줄 레이아웃
- **결과**: 영상 영역 **+25% 증가**

### 2. 사용자 경험 개선
- YouTube 소리 활성화 → 몰입감 증가
- 상품 카드와 결제 버튼을 한 줄에 배치 → 직관성 증가
- 채팅 간격 축소 → 더 많은 메시지 표시 가능

### 3. UI 효율성
- 불필요한 "상품" 버튼 제거
- 상품 카드를 클릭 가능한 버튼으로 통합
- 공간 활용도 극대화
