# 카카오 로그인 설정 체크리스트

## 🎯 현재 상황
- REST API 키: `5dd74bccb797640b0efd070467f3bafd` ✅ (올바름)
- 에러: KOE006 (앱 관리자 설정 오류)

---

## 📋 설정 확인 순서

### 1단계: 카카오 로그인 활성화 확인

**위치:** Kakao Developers Console > 왼쪽 메뉴 > "카카오 로그인"

**확인 사항:**
```
[ ] 활성화 설정 스위치가 ON (초록색)
[ ] 스위치가 OFF(회색)이면 클릭해서 ON으로 변경
```

**스크린샷:**
```
┌─────────────────────────────────┐
│ 카카오 로그인                    │
│ 활성화 설정    [OFF]  [●─] ON  │  ← ON이어야 함
└─────────────────────────────────┘
```

---

### 2단계: Redirect URI 등록 확인

**위치:** 카카오 로그인 페이지 아래쪽

**등록해야 할 URI:**
```
✅ https://live.ur-team.com/auth/kakao/callback
✅ http://localhost:3000/auth/kakao/callback
```

**확인 사항:**
```
[ ] "Redirect URI" 섹션 찾기
[ ] "등록" 또는 "+ 추가" 버튼 클릭
[ ] URI 정확히 입력 (대소문자, 슬래시 주의)
[ ] "저장" 클릭
```

**⚠️ 주의:**
- `https://` 포함해야 함
- 끝에 `/` 없음
- `callback` 소문자

---

### 3단계: 플랫폼 등록 확인

**위치:** 왼쪽 메뉴 > "플랫폼"

**등록해야 할 정보:**
```
플랫폼: Web
사이트 도메인: https://live.ur-team.com
```

**확인 사항:**
```
[ ] "Web 플랫폼 등록" 버튼 클릭
[ ] 사이트 도메인: https://live.ur-team.com 입력
[ ] "저장" 클릭
[ ] 플랫폼 목록에 "Web" 표시되는지 확인
```

---

### 4단계: 동의항목 설정

**위치:** 카카오 로그인 > "동의항목" 탭

**설정해야 할 항목:**
```
[ ] 닉네임: 필수 동의 또는 선택 동의
[ ] (선택) 카카오계정(이메일): 선택 동의
[ ] (선택) 프로필 사진: 선택 동의
```

---

## ✅ 최종 확인

모든 설정 완료 후:

```
1. [ ] 카카오 로그인 활성화: ON
2. [ ] Redirect URI 2개 등록됨
3. [ ] Web 플랫폼 등록됨
4. [ ] 동의항목 설정됨
5. [ ] 5분 기다림
6. [ ] 브라우저 캐시 삭제
7. [ ] 다시 테스트: https://live.ur-team.com/auth/kakao
```

## 🔍 만약 설정이 이미 되어 있다면?

### 앱 상태 확인
**위치:** 앱 설정 > 일반

**확인:**
- 앱 상태가 "개발 중" 또는 "서비스 중"
- "비활성" 또는 "중지됨"이면 활성화 필요
