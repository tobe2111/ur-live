## 🤔 추가로 구현 가능한 기능

### 1. 인플루언서 회원가입 기능 ⭐⭐
**현재**: 관리자가 수동으로 인플루언서 계정 생성  
**개선**: 인플루언서가 직접 회원가입 → 관리자 승인

```typescript
POST /api/seller/register
Body: {
  username, password, name, email, phone,
  business_name, business_number, bank_account
}
Response: { success: true, message: "승인 대기 중" }
```

**필요한 이유**:
- 인플루언서가 자유롭게 가입 가능
- 관리자는 승인/거부만 하면 됨

---

### 2. 인플루언서 프로필 페이지 ⭐
**현재**: 프로필 정보가 sellers 테이블에만 존재  
**개선**: 공개 프로필 페이지 생성

```
/influencer/:username
- 인플루언서 소개
- 예정된 라이브 목록
- SNS 링크 (Instagram, YouTube, Facebook)
- 과거 라이브 다시보기
```

**필요한 이유**:
- 팬들이 특정 인플루언서 팔로우 가능
- 브랜드 구축

---

### 3. 알림 시스템 ⭐⭐
**현재**: 알림 없음  
**개선**: 주요 이벤트 알림

**인플루언서 알림**:
- 새 주문 발생
- 재고 부족 (10개 이하)
- 라이브 시작 10분 전
- 환불 요청

**고객 알림**:
- 주문 상태 변경 (결제완료 → 배송중 → 배송완료)
- 관심 인플루언서 라이브 시작

**구현 방법**:
- 이메일 (SendGrid API)
- 카카오톡 알림톡 (필요 시)

---

### 4. 매출 대시보드 상세화 ⭐
**현재**: 기본 통계만 표시  
**개선**: 상세한 매출 분석

```
/seller/analytics
- 일별/주별/월별 매출 그래프
- 상품별 판매량
- 라이브별 매출 비교
- 시청자 수 vs 구매 전환율
```

**필요한 이유**:
- 데이터 기반 의사결정
- 어떤 상품이 잘 팔리는지 확인

---

### 5. 고객 문의 시스템 ⭐
**현재**: 없음  
**개선**: 고객이 인플루언서에게 직접 문의

```typescript
POST /api/seller/inquiries
Body: { product_id, message }

GET /api/seller/inquiries
→ 답변 대기 중인 문의 목록
```

**필요한 이유**:
- 상품 문의 처리
- 고객 만족도 증가

---

### 6. 리뷰 시스템 ⭐
**현재**: 없음  
**개선**: 구매자가 리뷰 작성

```typescript
POST /api/products/:id/reviews
Body: { rating, comment }

GET /api/products/:id/reviews
→ 상품 리뷰 목록
```

**필요한 이유**:
- 신뢰도 증가
- 다른 고객의 구매 결정 도움

---

### 7. 쿠폰 시스템 ⭐⭐
**현재**: 없음  
**개선**: 인플루언서가 쿠폰 발행

```typescript
POST /api/seller/coupons
Body: {
  name: "첫 구매 10% 할인",
  discount_type: "percentage", // or "fixed"
  discount_value: 10,
  min_order_amount: 50000,
  expires_at: "2026-03-01"
}
```

**필요한 이유**:
- 프로모션 진행
- 신규 고객 유치

---

### 8. 재고 자동 알림 ⭐
**현재**: 수동으로 재고 확인  
**개선**: 재고 부족 시 자동 알림

```typescript
// 재고가 10개 이하일 때
→ 이메일 발송: "상품 '{name}'의 재고가 부족합니다 (현재: {stock}개)"
```

---

### 9. 라이브 예약 시간 자동 알림 ⭐
**현재**: 수동으로 라이브 시작  
**개선**: 예약 시간 10분 전 알림

```
예약 시간 10분 전:
- 인플루언서에게 이메일 발송
- "라이브 시작 준비 해주세요!"
```

---

### 10. 고객 배송지 여러 개 저장 ✅ (이미 구현됨!)
