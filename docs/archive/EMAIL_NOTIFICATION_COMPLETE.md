## 📋 테스트 방법

### 방법 1: 셀러 대시보드에서 테스트
1. https://live.ur-team.com/seller/login 접속
2. 셀러 계정 로그인
3. "라이브 생성" 클릭
4. 라이브 정보 입력 후 저장
5. **jiwon@ur-team.com 메일함 확인** ✉️

### 방법 2: API 직접 호출 (개발자)
```bash
curl -X POST https://live.ur-team.com/api/seller/streams \
  -H "X-Session-Token: {셀러_세션_토큰}" \
  -H "Content-Type: application/json" \
  -d '{
    "title": "테스트 라이브",
    "description": "테스트입니다",
    "youtube_url": "https://youtube.com/watch?v=VIDEO_ID",
    "status": "live"
  }'
```

## ⚠️ 주의사항

### 발신자 이메일
- 현재: `noreply@live.ur-team.com`
- 실제로 존재하지 않는 주소여도 작동
- 스팸 필터에 걸릴 가능성 있음 (낮음)

### 스팸 방지 (선택사항)
더 나은 도달률을 위해:
1. Cloudflare Email Routing 설정
2. SPF 레코드 추가 (DNS)
3. DKIM 레코드 추가 (DNS)

**현재는 이대로도 충분합니다!**
