# 전자세금계산서 시스템 체크리스트

## 📋 사전 확인 사항

### 비즈니스 모델 확인
- [ ] **누가 발행?**: 인플루언서가 공급자로 발행 ✅
- [ ] **언제 발행?**: 배송완료 후 자동 발행 ✅
- [ ] **누구에게?**: 사업자 고객에게만 발행 (선택적) ✅
- [ ] **수수료 처리**: 
  - 공급가액 = 상품금액 - 플랫폼 수수료 10%
  - 예) 상품 129,000원 → 공급가액 116,100원 (90%) + 부가세 11,610원

### 법적 요구사항
- [ ] 인플루언서 사업자등록 필수
- [ ] 사업자등록증 확인 필요
- [ ] 발행 기한: 공급일 익일까지
- [ ] 국세청 전송 의무
