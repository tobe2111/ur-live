# 🎨 AI 디자이너에게 보낼 최종 프롬프트

## 📋 버전 1: 간단 버전 (GPT/Claude 등)

```
주문/결제 페이지 디자인 리뉴얼 요청

**배경**:
- 라이브 커머스 웹사이트의 결제 페이지
- 현재 URL: https://live.ur-team.com/checkout
- 기술: React + TypeScript + TailwindCSS
- 결제: 토스페이먼츠 위젯 SDK 사용

**현재 구조**:
```
┌─────────────────────────────────────┐
│ 헤더: ← 뒤로가기 | 주문/결제       │
├──────────────────┬──────────────────┤
│ 좌측 (2/3)       │ 우측 (1/3)      │
│ 1. 배송지 정보   │ 결제 금액 요약  │
│ 2. 주문 상품     │ - 상품금액      │
│ 3. 결제 수단     │ - 배송비        │
│   (토스위젯)     │ - 총금액        │
│                  │ [결제하기]      │
└──────────────────┴──────────────────┘
```

**디자인 요구사항**:
1. 모던하고 깔끔한 UI
2. 모바일 우선 반응형 (320px~)
3. 카드 기반 레이아웃 (shadow, rounded-lg)
4. 브랜드 컬러: 파란색 (#3B82F6)
5. 금액 숫자 크고 강조
6. 결제하기 버튼 눈에 띄게
7. 아이콘 활용 (lucide-react)

**절대 수정 금지**:
- `<div id="payment-method">` 영역 (토스페이먼츠 위젯)
- `<div id="agreement">` 영역
- 기능 로직 (버튼 onClick 등)

**참고**: 쿠팡, 마켓컬리, 토스 결제 페이지 스타일

TailwindCSS만 사용해서 JSX 코드를 작성해줘. 
기존 CheckoutPage.tsx 파일의 return 부분만 리뉴얼해줘.
```

## 📋 버전 2: 상세 버전 (전문 디자이너용)

```
**프로젝트**: 라이브 커머스 플랫폼 결제 페이지 UI 리뉴얼

**목표**: 기능은 유지하면서 UI/UX만 개선

**현재 페이지**:
- URL: https://live.ur-team.com/checkout
- 스택: React + TypeScript + TailwindCSS
- 결제: 토스페이먼츠 결제위젯 SDK

**페이지 구성**:

1. **헤더 영역**
   - 뒤로가기 버튼 (← 장바구니로 돌아가기)
   - 페이지 제목 (주문/결제)
   
2. **메인 콘텐츠** (좌우 2단 레이아웃)
   
   **좌측 (lg:col-span-2)**:
   
   A. 배송지 정보 카드
   - 수취인 이름
   - 전화번호
   - 주소 (도로명 + 상세주소)
   - 우편번호
   - [변경하기] 버튼
   
   B. 주문 상품 카드
   - 판매자별 그룹핑
   - 상품명, 옵션, 수량, 가격
   - 배송비 정보
   - 무료배송 조건 표시
   
   C. 결제 수단 카드
   - 🚨 토스페이먼츠 위젯 렌더링 영역
   - `<div id="payment-method" style="min-height: 350px">`
   - `<div id="agreement">`
   - **이 영역은 절대 수정 금지!**
   
   **우측 (lg:col-span-1, sticky)**:
   - 결제 금액 요약
   - 상품 금액
   - 배송비
   - 총 결제 금액 (강조)
   - [결제하기] 버튼 (크고 눈에 띄게)

**디자인 가이드**:

**색상**:
```css
Primary: #3B82F6 (버튼, 강조)
Success: #10B981 (무료배송)
Gray: #6B7280 (부가정보)
Background: #F9FAFB
White: #FFFFFF (카드)
```

**타이포그래피**:
- 제목: 24px~32px, font-bold
- 부제목: 18px~24px, font-semibold
- 본문: 14px~16px
- 금액: 24px~28px, font-bold, text-blue-600

**카드 스타일**:
```tsx
className="bg-white rounded-lg border border-gray-200 p-6 shadow-sm hover:shadow-md transition-shadow"
```

**버튼 스타일**:
```tsx
className="w-full py-4 bg-blue-600 hover:bg-blue-700 text-white text-lg font-semibold rounded-lg transition-colors disabled:bg-gray-400"
```

**반응형**:
- 모바일 (<640px): 상하 스크롤, 결제 요약 하단 고정
- 태블릿 (640px~1024px): 좌우 레이아웃 시작
- 데스크톱 (>1024px): 좌측 2/3, 우측 1/3, 우측 sticky

**아이콘**:
- lucide-react 라이브러리 사용
- MapPin, Package, CreditCard, ArrowLeft 등

**애니메이션**:
- 카드 호버 시 섀도우 증가
- 버튼 호버 시 배경 진하게
- 로딩 스피너 회전
- 모달 슬라이드업

**중요 제약사항**:
1. 토스페이먼츠 위젯 영역은 절대 수정 불가
2. 기존 기능 로직 유지 (onClick, onChange 등)
3. TailwindCSS만 사용 (인라인 style 최소화)
4. 접근성 고려 (aria-label, role 등)

**참고 디자인**:
- 쿠팡: 깔끔한 카드 디자인
- 마켓컬리: 프리미엄 느낌
- 토스: 심플하고 직관적

**결과물 형식**:
- CheckoutPage.tsx의 return 부분 JSX 코드
- TailwindCSS 클래스만 사용
- 주석으로 각 섹션 설명
- 반응형 클래스 포함 (sm:, md:, lg:)

**목표**:
기능은 그대로, 비주얼만 프리미엄 라이브 커머스 플랫폼 수준으로 업그레이드!
```

## 📋 버전 3: 즉시 사용 가능한 짧은 버전

- 우측: 금액요약, 결제버튼

1. 모던하고 깔끔

3. 카드 기반 (shadow, rounded)
4. 파란색 브랜드 컬러
5. 금액 강조
6. 버튼 눈에 띄게

TailwindCSS로 JSX 코드 작성.
토스위젯 영역(<div id="payment-method">) 수정 금지.
```

## ⚠️ 주의사항

AI가 생성한 코드를 받으면:

1. **토스페이먼츠 위젯 영역 확인**:
   ```tsx
   <div id="payment-method" style={{ minHeight: '350px' }}></div>
   <div id="agreement"></div>
   ```
   이 부분이 그대로 있는지 확인!

2. **기능 로직 확인**:
   - onClick 핸들러 유지
   - state 변수 사용 확인
   - API 호출 코드 유지

3. **반응형 테스트**:
   - 모바일 (320px)
   - 태블릿 (768px)
   - 데스크톱 (1280px)

4. **접근성 확인**:
   - 버튼 클릭 가능
   - 키보드 네비게이션
   - 스크린 리더 호환
