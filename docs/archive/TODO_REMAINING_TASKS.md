## 🎯 남은 페이지 및 우선순위

### 우선순위 1️⃣: 사용자 핵심 기능 (KREAM 스타일 적용 필요)

#### 1. **LoginPage** (`/login`)
- **현재 상태**: 기본 스타일
- **필요 작업**: 
  - KREAM 스타일 로그인 폼
  - 소셜 로그인 버튼 (카카오)
  - 미니멀 디자인
- **우선순위**: ⭐⭐⭐⭐⭐ (매우 높음)
- **예상 시간**: 2-3시간

#### 2. **SearchPage** (`/search`)
- **현재 상태**: 기본 검색 UI
- **필요 작업**:
  - KREAM 스타일 검색 바
  - 검색 결과 그리드 레이아웃
  - 필터 기능 (가격, 카테고리 등)
- **우선순위**: ⭐⭐⭐⭐ (높음)
- **예상 시간**: 3-4시간

#### 3. **MyOrdersPage** (`/my-orders`)
- **현재 상태**: 기본 주문 내역 리스트
- **필요 작업**:
  - KREAM 스타일 주문 카드
  - 주문 상태별 필터
  - 주문 상세 정보 확장
- **우선순위**: ⭐⭐⭐⭐ (높음)
- **예상 시간**: 4-5시간

---

### 우선순위 2️⃣: 판매자 페이지 (기능 개선 필요)

#### 4. **SellerPage** (`/seller`)
- **현재 상태**: 기본 대시보드
- **필요 작업**:
  - 매출 통계 차트 (Recharts 활용)
  - 최근 주문 목록
  - 빠른 액션 버튼
- **우선순위**: ⭐⭐⭐ (중간)
- **예상 시간**: 5-6시간

#### 5. **SellerProductsPage** (`/seller/products`)
- **현재 상태**: 제품 목록
- **필요 작업**:
  - 제품 그리드 레이아웃
  - 빠른 수정/삭제 기능
  - 재고 관리 UI
- **우선순위**: ⭐⭐⭐ (중간)
- **예상 시간**: 3-4시간

#### 6. **SellerOrdersPage** (`/seller/orders`)
- **현재 상태**: 주문 관리
- **필요 작업**:
  - 주문 상태별 탭
  - 배송 처리 UI
  - 주문 상세 모달
- **우선순위**: ⭐⭐⭐ (중간)
- **예상 시간**: 4-5시간

#### 7. **SellerLiveControlPage** (`/seller/live-control`)
- **현재 상태**: 라이브 방송 제어
- **필요 작업**:
  - 실시간 시청자 수
  - 채팅 모니터링
  - 제품 추가/수정 UI
- **우선순위**: ⭐⭐⭐ (중간)
- **예상 시간**: 5-6시간

---

### 우선순위 3️⃣: 관리자 페이지 (내부 도구)

#### 8. **AdminPage** (`/admin`)
- **현재 상태**: 기본 관리자 대시보드
- **필요 작업**:
  - 전체 매출 통계
  - 사용자 관리
  - 판매자 승인
- **우선순위**: ⭐⭐ (낮음)
- **예상 시간**: 6-8시간

#### 9. **AdminSettlementPage** (`/admin/settlement`)
- **현재 상태**: 정산 관리
- **필요 작업**:
  - 정산 내역 테이블
  - 필터 및 검색
  - 엑셀 다운로드
- **우선순위**: ⭐⭐ (낮음)
- **예상 시간**: 4-5시간
