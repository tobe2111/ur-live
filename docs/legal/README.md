# 유통스타트 계약서 (B2B)

가입 시 체결하는 유통스타트 표준 계약서 2종. 경쟁사(셀파이 등) 문구 복사가 아닌, **유통스타트 실제 운영 모델 기반 자체 작성**.

| 대상 | 파일 | 당사자 | 핵심 |
|---|---|---|---|
| **판매사**(유통회원) | [`판매사-물품공급계약서.md`](./판매사-물품공급계약서.md) | 유통스타트 ↔ 판매사 | 공급가로 사입→재판매, **예치금 선불결제**, 권장가 준수, 제조사 직발송 |
| **제조사**(공급사) | [`제조사-공급계약서.md`](./제조사-공급계약서.md) | 유통스타트 ↔ 제조사 | 상품 등록→**검수·승인**, 공급가+**플랫폼 마진**, 정산 성숙(반품 안전창), 전자세금계산서 |

## 우리 모델이 계약서에 반영된 부분
- **3주체**: 제조사(공급) → 유통스타트(검수·마진·정산) → 판매사(사입·재판매).
- **가격**: 제조사가 **공급가(부가세 포함)** 입력 → 유통스타트가 **유통 마진**(상품별 조정) 더해 판매사 공급가 산정 → 제조사 정산 = 공급가 전액, 마진 = 유통스타트 수익.
- **결제**: 판매사는 **예치금(선불 충전)** 으로 결제(외상·사이트외 결제 불인정). 환불 시 예치금 복원.
- **정산(제조사)**: 결제완료분 적립 → **반품 가능 기간(안전창) 경과 후 확정·지급**, 환불 시 역전. 정산 주기는 사이트 고지(예: 주 단위 — 구체 주기 확정 시 반영).
- **배송**: 제조사 **직발송**(택배), 송장 등록.
- **등급**: 회원등급은 노출·혜택 차등(가격 차별이 아닌 노출 큐레이션).
