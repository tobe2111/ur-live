## 📋 다음 구현 사항 (우선순위별)

### 🔥 **Priority 1: 셀러 라이브 스트림 관리 UI** (HIGH)

**현재 상태**: 
- ✅ API는 완성 (`POST /api/seller/streams`, `PUT /api/seller/streams/:id`)
- ❌ UI가 없음 (셀러가 사용 불가)

**필요한 작업**:
1. **셀러 대시보드에 라이브 관리 페이지 추가**
   - 라이브 스트림 목록 조회
   - 새 라이브 생성 폼
   - 기존 라이브 수정 폼

2. **라이브 생성 폼 구현**
   ```tsx
   // 필요한 필드
   - 제목 (title)
   - 설명 (description)
   - YouTube URL (youtube_url) // 자동으로 Video ID 추출
   - 상태 (status: 'scheduled' | 'live' | 'ended')
   - 예약 시간 (scheduled_at)
   - SNS 링크 (Instagram, YouTube, Facebook)
   ```

3. **라이브 관리 기능**
   - 실시간 미리보기
   - 라이브 시작/종료 버튼
   - 현재 상품 설정
   - 시청자 수 표시 (선택)

**예상 시간**: 2-3시간  
**우선순위**: 🔥 **가장 중요!** (셀러가 직접 라이브를 관리할 수 있어야 함)
