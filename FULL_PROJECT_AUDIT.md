## 🚨 **발견된 누락 사항 (우선순위별)**

### 🔴 **P0 - 크리티컬 (비즈니스 블로커)**

#### 1. **실제 결제 시스템** ❌
**현재 상태**: Mock 결제만 가능 (가짜 돈)
**문제**: 실제 매출 발생 불가능
**필요 작업**:
- NicePay API 키 발급 및 적용
- Toss Payments 연동 (대안)
- 결제 승인 API 완성
- 결제 실패 처리
- 환불 시스템

**시간**: 1일
**임팩트**: 🔥 매우 높음 (매출 직결)

---

#### 2. **주문 내역 페이지** ❌
**현재 상태**: 백엔드 API만 있음, UI 없음
**문제**: 사용자가 주문 확인 불가
**필요 작업**:
- `/my-orders` 페이지 생성
- 주문 목록 UI
- 주문 상세 정보
- 배송 추적
- 취소/환불 요청

**시간**: 2시간
**임팩트**: 🔥 높음 (사용자 경험)

---

#### 3. **셀러 주문 관리** ⏳
**현재 상태**: 부분 구현 (조회만 가능)
**문제**: 셀러가 배송 처리 불가
**필요 작업**:
- 주문 상태 변경 (준비중 → 배송중 → 완료)
- 송장 번호 입력
- 주문 상세 조회
- 배송 알림 (선택)

**시간**: 3시간
**임팩트**: 🔥 높음 (셀러 운영)

---
